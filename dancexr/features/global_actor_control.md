## Overview
Global Actor Control provides scene-wide scaling settings that apply uniformly to all loaded actors.

## Settings

* **Actor Scale** (0.15–5.0): Scales the size of all actors in the scene globally. The default value is 1. Use this to uniformly make all actors larger or smaller without adjusting them individually.
* **Motion Scale** (0.5–1.5): Scales the position offsets of motions globally across all actors. Adjusting this value changes how far actors move during playback while keeping their size unchanged.
