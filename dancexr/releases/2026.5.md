# Release 2026.5

## Introducing DanceXR Operator

### AI chat overhaul
AI chat has been rebuilt to adopt an agentic workflow for character behavior simulation, and using Operator as dedicated backend allow it to run on a standard gaming setup without needing a separate server or cloud connection. 
- **Environment awareness**: New chat system access scene information to place the character in current context, they are aware of the time, lighting, background and stage changes. 
- **Memory and persistence**: The characters now have a better memory system that help them keep track of their feelings, relationships and key events instead of replaying past dialogues. 
- **High quality TTS**: The new TTS system has better voice quality, more languages support and still fast enough for real-time interaction.
