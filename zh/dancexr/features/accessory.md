# 配饰

可将道具和对象附着到角色模型上的特定骨骼。

可用的附件点包括：**Pole**、**左手**、**右手**、**胸部**、**头部**、**左脚**和**右脚**。

每个附件都有其独立的面板，用于模型选择、尺寸和对齐、表面材质、运动摆动和X光剖面渲染。有关每个子面板的详细信息，请参阅嵌套的附件文档。

**对称双手** 将所有设置从左手附件复制到右手中，因此您只需配置一次。
**对称双脚** 对脚附件执行相同的操作。

# 子组件

## Pole

配置附着到角色特定骨骼上的道具（模型或程序化几何体）。支持带运动、X光剖面和表面着色的立柱对象、手持物品和解剖学道具。

**模型** 用于选择加载的配件或使用默认的程序化几何体。锚点偏移、配件配置和表面设置可在嵌套的子面板中找到。

**运动** 为附件应用上下摆动。**拉手** (pole 模式) 会将附近的双手拉向立柱表面；**抓取姿势** 会自动调整手部抓握；**手部运动** 可相对于附件的移动来偏移双手。

**X光** 通过道具渲染一个半透明的切面。**强度** 控制可见性，而 **半径**、**高度**、**偏移** 和 **颜色** 定义了圆柱形的形状和色调。**透明度** 调整整体材料的透明度。

### 锚点偏移

在应用任何附件偏移之前，精细调整锚点骨骼的位置和旋转。 **位置** 和 **旋转** 都是微小调整（±1 个单位，±90 度），用于补偿模型之间骨骼的差异。

### 尺寸与对齐

控制附件道具的物理尺寸和空间对齐。 **物体半径** 和 **物体长度** 定义了程序化几何体的尺寸（例如杆）。 **缩放** 是加载模型的对数倍增益。

**方向** 用于选择默认朝向轴（Y/X/Z 上/下）。

**偏移** 和 **旋转** 在方向之后应用局部空间调整。**吉他模式** 会旋转道具，使其跟踪手部位置，就像拨弦一样。

### 运动

驱动附件道具进行有节奏的上下摆动，并与音乐节拍同步。开关启用运动；**距离** 设置移动范围；**角度** 控制在最大延伸时的倾斜度。

嵌套的速度配置定义了节拍曲线和时间模式。
