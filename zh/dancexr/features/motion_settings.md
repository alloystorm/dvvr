# 运动设置

控制如何将动作数据（VMD、BVH 等）应用于角色模型，包括缩放、时间偏移、IK 处理和姿势校正。


## 运动参数

**运动缩放** 会乘数化动作文件中所有的位置移动——当动画夸张或对于模型的比例来说过于微妙时很有用。**垂直位置缩放** 可以独立地缩放中心骨骼的上下移动，以减少晃动或跳跃。

**镜像** 会翻转左右的运动数据，主要用于为设计反向主导性的 VMD 文件。**时间偏移百分比** 和 **时间偏移秒数** 调整了角色在动画时间轴上开始播放的时间——这对于同步多个角色或补偿动作引导（lead-in）很有用。

**腿部角度** 为腿部骨骼增加了微小的旋转偏差，从而调整整个动画中的站姿宽度或脚尖朝外角度。


## IK 设置

**继承骨骼** 遵循 PMX 文件中定义的继承父级约束（例如手部骨骼跟随手臂骨骼）。**运动腿部 IK** 在处理运动时应用了一个通用的双骨骼腿部 IK 流程，即使模型没有专用的 IK 骨骼也能实现脚部着地。**模型 IK** 使用 PMX 本身定义了的 IK 骨链。当模型 IK 与运动 IK 冲突时，请尝试禁用其中一个。


## 旁观者

将角色标记为旁观者——他们被排除在阵型模式和灯光分配之外，对于观众或背景角色很有用。


## 姿势调整

在应用任何运动之前，旋转默认的 T 姿骨骼角度。请查看嵌套面板以获取每个骨组的旋转控制——这是修正导入的静止姿势与动画假设不匹配模型的首要方法。


## 可视化

**显示虚拟骨骼**、**可视化骨骼** 和 **可视化 IK 目标** 会渲染骨骼层次结构和 IK 目标位置的调试小组件（仅限 PMX 模型）。


# 子组件

## sd_pose

调整角色每个主要骨组的默认（零位）旋转。这些数值会让骨骼偏离其导入的 T 姿，从而可以修正那些站立时手臂过宽、腿部过近或手指张开不当的模型。

每个设置都是一个三轴旋转（X/Y/Z），单位是度。左侧的骨骼会自动镜像到右侧。**无名指** 和 **中指** 是相对于小指和食指的标量乘数，保持了手指弯曲的比例。
