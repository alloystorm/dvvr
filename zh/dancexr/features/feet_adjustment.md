# 脚部调整

确保脚能够动态适应地面高度，实现逼真的位置和移动。

**地面偏移** 调整垂直参考值。

**脚部贴地** 面板可使脚保持在地面：
**强度**（总体的程度，0禁用）、**脚底高度**（脚趾到地面间隙）、**跳跃高度**（暂停校正的空中阈值）、**身体平滑度**和**脚部平滑度**（对身体和单个足部的响应平滑处理），**最大足倾角**（足部可倾斜以在地面设置接触的最大角度）以及**触及倾角**（当腿无法触及时保持接触的额外趾尖下沉）。

**高跟** 面板将脚抬升到 **高跟** 姿态。

**减少滑动** 可固定一个已着地的足部，以停止“滑行”：启用它，然后设置**强度**（锁定程度）和**滑动距离**（在释放前允许漂移的远度，防止腿部过度拉伸）。

**脚角** 添加手动 **旋转 X**、**Y** 和 **Z** 偏移。
