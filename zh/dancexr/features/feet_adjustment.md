# 脚部调整

确保脚能够动态适应地面高度，实现逼真的位置和移动。

**地面偏移** 调整垂直参考值。**脚部贴地** 面板控制**强度**（拉向地面）、**脚底高度**（脚趾到地面距离）、**身体抬升**（COM修正）、**接触弯曲脚趾**（脚趾屈曲）、**接触旋转**（旋转惯性）、**跳跃检测**（空中抑制）以及**跳跃高度**阈值。

**虚拟鞋子** 面板通过**内侧**、**外侧**、**前方**、**后方**偏移、**鞋子距离**、**脚跟顶部**高度、**模式**（线框、不透明、镀铬、实心玻璃、薄玻璃）、**平滑度**和**颜色**可视化脚部接触。

**鞋跟** 面板模拟高跟姿态，包含**高跟**高度、**限制角度**、**高跟脚趾弯曲**和**高跟混合**，用于动画混合。

**减少滑动** 应用摩擦力，通过**距离阈值**和**阻尼**控制。**脚角** 增加了手动**旋转 X**、**Y** 和 **Z** 偏移。
