# 动作覆盖

用于定制和增强演员动作的高级工具。它与骨架系统集成，可以精确控制身体、头部、手部和腿部运动。

**预设** 包括 自由、摇摆动作、悬浮摩托车、摇马、杆状动作和杆状混合。**身体** 面板控制 **位置** 和 **旋转** 锁、**阻尼**、**倾斜**、**弯曲**、**高度** 和 **前后** 偏移。**摇摆动作** 模拟具有可调 **角度**、**上下**、**前后**、**深度变化** 和 **脚部运动** 参数的节奏运动。

**骑乘模型** 面板会生成像悬浮摩托车或马鞍等道具，并提供 **加速**、**阻力**、**转弯时倾斜**、**位置**、**旋转**、**缩放** 和 **粒子效果** 控制。**头部姿势** 为颈部和头部增加了旋转偏移。**腿部姿势** 动态调整脚部放置。**手部姿势**（左和右）支持预设和对称开关。

用例包括限制舞蹈动作范围、创建摇马动画，以及使用骑乘模型和粒子轨迹将舞蹈转换为类似车辆的运动。

# 子组件

## 身体

用于身体姿势调整的嵌套配置。**位置** 和 **旋转** 开关锁定特定轴。**阻尼** 使位置跟踪更平滑。**倾斜**、**弯曲**、**扭转** 和 **头部** 用于旋转躯干和颈部。**高度** 和 **前后** 用于平移身体。**距离** 模式在 **检测范围** 内保持接近 **目标角色** 的距离，强制执行 **最小** 和 **最大距离** 限制。

## 摇摆动作

用于节奏摇摆动作模拟的嵌套配置。**摇摆角度** 设置弧度，**上下** 和 **前后** 控制平移幅度。**深度变化** 和 **最大深度** 控制接近度的振荡。**脚部运动** 将脚部移动与摇摆周期混合。包含一个用于定时控制的 **运动速度** 子配置。

## 头部姿势

用于应用于颈部和头部骨骼的头部姿势旋转偏移的嵌套配置。控制 **旋转 X**、**Y** 和 **Z** 角度。

## 骑乘模型

用于可骑乘道具模型（悬浮摩托车、马鞍）的嵌套配置。**模型** 选择道具类型。**加速** 和 **阻力** 控制前向运动物理。**转弯时倾斜** 在方向改变时增加倾斜。**位置**、**旋转** 和 **缩放** 调整道具的附着。**粒子效果** 启用排气火花特效。计算带阻力和倾斜滤波的速度运动。
