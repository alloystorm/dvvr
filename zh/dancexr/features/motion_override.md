# 动作覆盖

高级运动控制系统，通过骨架调整覆盖和增强演员的动作。对于限制舞蹈范围、创建车辆模拟或添加摇摆等节奏动画非常有用。

## 预设

快速启动配置包括 **自由**（最少覆盖）、**摇摆动作**、**悬浮摩托车**、**摇马**、**杆状动作** 和 **杆状混合**。每个预设都会自动启用并配置相关的子面板。

## 身体控制

**位置锁定** 限制沿水平、垂直或两个轴的运动——这对于让角色保持在画面内很有用。**旋转锁定** 可防止躯干在跳舞时转动。**阻尼** 使得位置变化更加平滑（数值越高，响应越慢）。**倾斜**、**弯曲** 和 **扭转** 用于旋转躯干，而 **高度** 和 **前后** 则平移身体。**距离** 模式保持与目标角色设定的距离范围。

## 摇摆动作

模拟节奏运动，包含 **角度**（弧度宽度）、**上下** 和 **前后**（平移幅度）以及 **深度变化**（接近度振荡）。**脚部运动** 将脚部移动与摇摆周期混合。计时通过 **运动速度** 子配置控制。

## 骑乘模型

使用 **加速度** 和 **阻力** 物理效果附着道具（悬浮摩托车或马鞍）。**转弯时倾斜** 在方向改变时增加倾斜。调整 **位置**、**旋转** 和 **缩放** 使道具适合演员。启用 **粒子效果** 用于尾气火花特效。

## 手部和腿部姿势

**手部姿势**（左/右）提供手势预设和自定义手指控制，包含用于抓握或指向的 IK 模式。**腿部姿势** 调整脚部放置，并支持对称性。**头部姿势** 添加旋转偏移。

# 子组件

## 身体

用于身体姿势调整的嵌套配置。**位置** 和 **旋转** 开关锁定特定的轴。**阻尼** 使位置跟踪更平滑。**倾斜**、**弯曲**、**扭转** 和 **头部** 用于旋转躯干和颈部。**高度** 和 **前后** 用于平移身体。**距离** 模式在 **检测范围** 内保持接近 **目标角色** 的距离，强制执行 **最小** 和 **最大距离** 限制。

## 摇摆动作

用于节奏摇摆动作模拟的嵌套配置。**摇摆角度** 设置弧度，**上下** 和 **前后** 控制平移幅度。**深度变化** 和 **最大深度** 控制接近度振荡。**脚部运动** 将脚部移动与摇摆周期混合。包含一个用于定时控制的 **运动速度** 子配置。

## 头部姿势

应用于颈部和头部骨骼的头部姿势旋转偏移的嵌套配置。控制 **旋转 X**、**Y** 和 **Z** 角度。

## 腿部姿势

用于腿部和脚部放置的嵌套配置。**左** 和 **右** 子面板可调整 **脚部 X/Y/Z** 位置、**脚部旋转 X/Y/Z** 角度、**脚趾** 弯曲和 **展开**（腿部张开）。**对称** 链接两侧。**相对于地板** 基于地板平面而非髋部进行脚部偏移。**最大扭转角度** 限制了躯干角度偏离腿部的程度。预设：*坐*、*骑*、*跪*、*站*。

## 左手

手部姿势控制的嵌套配置。选择一个 **手势** 预设或使用 **自定义姿势** 来调整单个手指弯曲。**位置** 和 **旋转** 使手在空间中偏移。**旋转类型** 设置坐标系（参考骨骼、自身、绝对或无）。**参考骨骼** 和 **参考角色** 将手固定到身体的另一个部分或另一个角色。**IK 模式** 在正常、圆柱、球体或对齐 IK 解算器之间切换。**混合手部动作** 根据距离淡化姿势。**对称** 链接左右姿势。

### 运动

驱动附着道具的节奏上下振荡，同步于音乐节拍。开关启用运动；**距离** 设置移动范围；**角度** 控制最大延伸时的倾斜度。嵌套的速度配置定义了节拍曲线和定时模式。

## 右手

手部姿势控制的嵌套配置。选择一个 **手势** 预设或使用 **自定义姿势** 来调整单个手指弯曲。**位置** 和 **旋转** 使手在空间中偏移。**旋转类型** 设置坐标系（参考骨骼、自身、绝对或无）。**参考骨骼** 和 **参考角色** 将手固定到身体的另一个部分或另一个角色。**IK 模式** 在正常、圆柱、球体或对齐 IK 解算器之间切换。**混合手部动作** 根据距离淡化姿势。**对称** 链接左右姿势。

### 运动

驱动附着道具的节奏上下振荡，同步于音乐节拍。开关启用运动；**距离** 设置移动范围；**角度** 控制最大延伸时的倾斜度。嵌套的速度配置定义了节拍曲线和定时模式。

## 骑乘模型

可骑乘道具模型（悬浮摩托车、马鞍）的嵌套配置。**模型** 选择道具类型。**加速度** 和 **阻力** 控制前向运动物理。**转弯时倾斜** 在方向改变时增加倾斜。**位置**、**旋转** 和 **缩放** 调整道具的附着。**粒子效果** 启用尾气火花 VFX。计算带阻力和倾斜滤波的速度运动。
