# 分离物体

将选定的骨骼从角色模型上分离，并应用独立的物理模拟，使配饰或持有的物体能够动态地掉落。

## 骨骼选择

使用 **骨骼选择** 来选择哪些骨骼将成为分离的物理对象。只能选择非运动学骨骼。

## 物理效果

**重力** 开关用于控制分离骨骼的重力。**质量** 控制刚体重量——越重的物体掉落得越快，推力也越大。**阻尼** 在一段时间内增加空气阻力，减缓运动。

## 碰撞体

选择碰撞形状：*无*、*球体* 或 *胶囊体*。**碰撞体半径** 设置球体/胶囊体的宽度。**碰撞体长度** 沿骨骼轴延伸胶囊体。
