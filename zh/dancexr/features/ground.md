# 地面

控制场景所在的地面和周围舞台——地面表面、程序化舞台和泳池几何体，以及（在 HDRP 上）水系统。一组预设将这些子功能打包成一键式的外观，例如 *木材*、*舞台*、*泳池*、*海洋* 和 *音频可视化*。

## 地面高度

舞台中心的垂直偏移量，单位为米。使用此设置可将地面与已跟踪的活动区域或捕获的场景对齐，确保演员不会陷进地面或漂浮在地面上方。其他锚点（灯光、连接到舞台的摄影机）将遵循此偏移量。

## 地面设置

定义地面圆盘本身的子组——包括其表面材质、半径和舞台感知可见性。有关详细信息，请参阅下方的 *地面* 组。

## 舞台几何体

包围地面圆盘的程序化舞台、跑道、墙壁和内部泳池池。有关详细信息，请参阅下方的 *舞台/泳池* 组。舞台几何体与地面圆盘组合——两者可以同时可见。

## 水系统 (HDRP)

仅适用于 HDRP 的水表面，锚定在舞台中心，用于海洋和泳池预设。有关详细信息，请参阅下方的 *水系统* 组。每当舞台几何体更改时都会重新应用，以便水面能够跟随泳池池。不支持 URP——此时水着色器全局参数将被清除。

## 预设

预设列表将子功能组合成命名外观。选择一个预设可以在一步内切换地面的开关、选择地面表面、选择舞台几何体和水体变体。预设是切换普通地面、跑道舞台、泳池、海洋、音频可视化地面、投影屏幕或 LED 盒子最快的方式。

# 子组件

## 地面

场景所在的平坦地面圆盘。切换禁用它会完全隐藏地面（当舞台道具或 AR *透视* 应该接管时非常有用）。**半径**设置了圆盘的大小——对于户外外观，足够大以填满可见地平线；对于更紧凑的室内场景，则较小。**如果存在舞台则隐藏** 自动在加载具有 *舞台* 目的的演员时抑制地面，从而防止物理舞台模型堆叠在程序化圆盘之上。

**表面**子组选择地面材质——纹理瓷砖、天空投影穹顶或纯色——并且它也与父级预设捆绑包共享其预设列表。

## 舞台/泳池

由跑道底板加上可选的外部墙壁、内部池和后墙构建的程序化舞台——用于跑道、泳池、房间、投影屏幕和 LED 盒子。内置预设涵盖了常见的形状；下方的字段允许您调整它们。

### 位置

**抬升**将舞台提高到地面上方或使其沉入地下；负的抬升会在地面圆盘上挖一个洞，以便内部池可以蓄水。**前后偏移**沿 Z 轴将整个舞台向前或向后移动，可用于与已跟踪的活动区域对齐。

### 几何体

**几何体**子组设置底板的尺寸。**中心宽度** / **中心深度** 定义了主矩形；**侧面 / 前部 / 后部延伸** 在其外部添加了跑道部分。**墙壁高度** 和 **墙壁厚度** 控制环绕周边的边缘——将墙壁高度设置为 0 可实现平齐平台。**后部高度** 会抬高舞台后方的垂直板，用于背景板和投影屏幕。**漂浮**将底板从地面几何体上分离，使其可以在水面上放置而不会发生 Z 轴冲突。

### 表面

三个独立的地面表面——*顶部*、*侧面* 和 *背景*——每个都有自己的纹理、平铺和颜色。这使得舞台顶部、环绕边缘和后板可以显示为不同的材质（例如，木质顶部搭配金属侧面）。

### 定制孔洞

默认情况下，挖出的孔洞遵循舞台轮廓。切换**定制孔洞**可使用 **左** / **右** / **前** / **后** / **顶** / **底** 的值显式覆盖边界——这对于非矩形切割或将孔洞与导入的场景对齐非常方便。

## 水系统

锚定在舞台中心的 HDRP 水表面。仅限 Pro 版。用于坐落在舞台内部池、静水湖和无限海洋中的泳池。预设将常见的外观打包在一起。

### 类型和高度

**类型**选择水体几何体。*泳池* 将表面尺寸定为舞台的跑道尺寸，当舞台有挖出的池时选择此项。*河流* 是覆盖地面圆盘的有限四边形。*海洋* 是无限的——可用于填充地平线、不受舞台限制的水域。

**高度**是相对于舞台中心的的水位。负值会将水放置在地面以下（匹配由负舞台 *抬升* 挖的泳池池）。

### 波浪

**涟漪**驱动小型风切波——对于平静的水面，请保持此值较低。**大浪**驱动定义海洋外观的宽阔涌浪；将其设置为 0 可获得像玻璃一样的平静表面。**波浪范围**控制湿效应向上达到与水接触材料的高度。

### 光学属性

**吸收距离**是指从上方观察水体时可见的距离；对于浑浊的水，应降低此值；对于清澈的水，应提高此值。**吸收乘数**在从水面下方观察时缩放该距离。**折射最大距离**限制水下的折射深度——更高的值会夸大失真。**焦散**设置了投影在水下几何体上的焦散图案的亮度。
