# 故障排除

修复常见的动画和骨架绑定问题。**将身体旋转应用于中心** 将臀部和躯干的旋转传输到中心骨骼。**扭曲校正** 可减少手臂和腿部关节的扭曲，并设有独立的**上臂**、**前臂**和**腿部扭曲**滑块。**肘部轴** 调整肘关节的旋转轴。**手部缩放** 调整手的大小。**BVH拇指运动** 减少 BVH 动画的拇指移动。**限制颈部/头部旋转** 限制上身的旋转。**重置过渡** 从标准姿势执行平滑的物理重置。**跳过运动学更新** 禁用未动画运动学骨骼的更新。
