# 远程访问

远程控制允许 **Android 应用**充当连接到同一本地网络内另一设备上运行的 DanceXR 的无线控制器。当主会话在大型屏幕（例如 PC 或 Quest）上播放时，您可以使用手机或平板电脑更改场景、动作和设置。


## 要求

* 已安装了 DanceXR Android 应用的 Android 设备。
* 一台正在同一本地局域网 (LAN) 上运行的 PC、Quest 或其他 DanceXR 主机。
* 网络必须允许设备进行本地发现（无客户端隔离）。


## 如何连接

1. 在主机设备上启用 **远程访问**。
2. 设置 **端口号** 和安全的 **密码**。
3. 打开 Android 应用，在主菜单中选择 **远程控制**，然后从已发现的设备列表中选择主机。
4. 按照提示输入密码，以同步菜单和触控板控件。
