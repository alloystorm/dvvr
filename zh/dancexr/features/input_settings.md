# 控制器输入

输入设置允许您为键盘/鼠标和游戏控制器（包括VR手柄）映射和校准控制器输入。


## 选项

**触控板模式**改变了控制器触摸板的行为。启用后，轴值仅在物理按下触控板按钮时输出。

**使用辅助轴**允许读取具有拇指摇杆和触摸板的控制器（例如WMR手柄）的输入。

**使用游戏手柄**即使在VR模式下也能启用游戏控制器/游戏手柄。

**使用握把输入**开关握把按钮输入。将其关闭可防止意外抓取或通过握把拖拽的交互。

**握把阈值**（0.0至1.0）设置了握把按钮的触发阈值。对于高灵敏度控制器（如Valve Index），这很有用，可以防止意外触发。


## 控制器映射

您可以自定义控制器上各种轴和按钮的映射：
* **轴**可以映射到相机移动、旋转、时间线拖动或帧步进等动作。
* **按钮**可以映射到播放动画、加载下一个/上一个模型、慢动作或重置物理等动作。
* **Alt键**可以分配以将键盘按键映射到控制器按钮。
