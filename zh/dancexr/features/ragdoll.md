# 布娃娃

使用基于粒子的布娃娃物理模拟取代演员的动画骨骼。身体会松弛，并对重力和外部力做出反应。

**弹簧力**和**阻尼**（两者均为对数形式）控制着布娃娃恢复到动画姿势的积极程度——低值会使其松垮，高值会使其更接近动画。**重力**用于切换施加在布娃娃身上的重力拉力。

**锁定左/右脚**、**锁定头部**和**锁定左/右手**将特定的身体部位固定在其动画位置，防止其松弛——当身体其余部分松弛时，这对于保持双脚着地非常有用。

**精细调整**设置每个骨骼的力尺度，范围从 0（完全松弛）到 1（完全刚性）。调整如腿、膝、臂、肩、躯干、颈和头等独立身体部位，以创建局部刚度——例如，保持躯干坚挺，同时让四肢自然下垂。
