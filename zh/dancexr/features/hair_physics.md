# 毛发物理效果

将链式物理效果应用于模型上的毛发发丝。选定的根骨骼会连接成一个物理链，每个子骨骼都由弹簧阻尼动力学驱动进行摆动和反弹。


## 骨骼选择

使用 **选择骨骼** 来挑选根毛发骨骼——系统会自动遍历子级骨骼来构建链。**跳过前 X 根骨骼** 会让根骨骼保持未连接状态，提供更牢固的基础。**最大层数** 限制了链的深度（0 = 无限制）。预设允许您在不同的模型上保存配置。


## 物理模式

**自动** 遵循系统级的默认设置。**PhysX** 使用基于关节的刚体物理，配有胶囊或球体碰撞体。**XPBD** 使用基于粒子的模拟，对长链更稳定。此模式决定了显示哪个设置面板。


## PhysX 设置

使用 PhysX 模式时可见。**弹簧力**（对数）控制刚度——值越高，毛发越靠近其静止姿态。**阻尼** 减少振荡。**刚度衰减** 柔化每个链级的弹簧，使发梢能更自由地飘动。**扭转限制** 约束绕骨骼轴的旋转。**限制力** 控制关节抵抗其约束的力度。**质量** 和 **阻力** 影响惯性和空气阻力。**碰撞体半径** 设置碰撞球体大小；**碰撞体长度** 沿着骨骼按胶囊进行缩放。**锚点位置** 选择关节连接的位置（0 = 父骨骼，1 = 子骨骼）。


## XPBD 链式结构

使用 XPBD 模式时可见。配置基于粒子的链式模拟，包含**旋转符合度**、**扭转符合度**、**刚度衰减**、**质量衰减** 和**约束阻尼**。**粒子锚点** 控制附着位置。碰撞层设置为 Hair，以便与场景其余部分进行交互。


## 可视化

**可视化体** 渲染物理体的碰撞体形状。
**可视化关节** 显示了关节限制和驱动目标作为线框显示。


# 子组件

## XPBD

配置用于毛发、布料和其他悬挂部件的基于粒子的链式或网格模拟。**旋转符合度** 控制链在每个关节处的弯曲程度（越高 = 越灵活）。**扭转符合度** 控制绕骨骼轴的旋转。对于网格模式，**侧向符合度** 在相邻链之间增加交叉连接。

**刚度衰减**（10的幂次刻度）乘以每个级别的符合度——高于 1 的值会使链向发梢逐渐松弛。**质量衰减**（2的幂次刻度）降低每个级别的质量。**粒子锚点** 将关节定位到链段上。**约束阻尼** 平滑求解器步骤之间的振荡。**惯性** 为运动变化增加阻力。**粒子半径** 设置毫米单位下的碰撞体大小。**使用球体形状** 用球体代替胶囊来调试。
