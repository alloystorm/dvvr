# 应用设置

全局应用程序设置，用于控制启动行为和内容处理。

## 启动

**自动加载** 控制应用程序启动时会发生什么。
*无* 以一个空的舞台开始，*最近角色* 加载最近使用的角色模型，*最近场景* 恢复上次保存的场景。

## 纹理加载

**翻转 DDS 压缩** 和 **翻转 DDS 未压缩** 控制 DDS 纹理在加载时是否会进行垂直翻转。如果纹理显示为颠倒状态，请启用这些设置——所需设置取决于纹理压缩格式。

## 用户界面行为

**辅助变换第三轴** 更改了在使用变换辅助工具时鼠标滚轮的行为。*旋转* 沿着选定的轴旋转物体，而 *深度* 则使物体更近或更远。

**使用翻译名称** 会在可用时显示本地语言的内容名称，而不是原始文件名。

**从保存数据恢复标签** 如果内容缓存被重建或损坏，它会尝试从保存的设置中恢复标签数据。
