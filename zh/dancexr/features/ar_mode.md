# AR设置

AR模式适用于支持的iOS、Android和Quest设备。并非所有设备都具备AR功能，因此，如果在菜单中看不到AR选项，这意味着您的设备不支持AR。

在iOS和Android上，程序总是以普通模式启动，不带AR。您可以通过转到场景菜单并勾选“AR”选项来启动AR。

在Quest上，应用程序支持透视（Passthrough）功能，可显示您的真实环境。

## 设置

**显示摄影机** 切换物理摄影机背景的显示开关。

**隐藏地面** 隐藏虚拟地面平面，使虚拟模型看起来直接站立在您真实的地面上。

**遮挡** 启用虚拟模型被现实世界物理对象遮挡的功能。仅在受支持的设备和平台上工作。

**双击重置场景**（非Quest）允许通过双击快速重置屏幕前的虚拟场景位置。

## VR选项（仅限Quest）

* **焦平面渲染级别 (Foveated Rendering Level)** 调整焦平面渲染级别，以提高VR性能。
* **可见手部 (Hand Visible)** 切换虚拟手部的渲染开关。
* **投射阴影 (Cast Shadow)** 控制手部是否投射阴影。
* **左右手姿势 (Left/Right Hand Pose)** 选择手部的默认姿势。
* **UI自动返回 (UI Auto Return)** 使用户界面跟随您的头部方向移动。
* **UI距离 (UI Distance)** 设置菜单的远近程度。
* **指针校准 (Pointer Calibration)**（方向、朝向、偏移）校准VR指针光线。
