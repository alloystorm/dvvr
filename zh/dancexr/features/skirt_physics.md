# 裙子物理

为模型身上的裙子或连衣裙网格添加物理模拟。支持多达 8 个骨骼群组，可配置关节、碰撞体和基于粒子的网格变形。

## 群组管理

**主群组**始终处于活动状态，并定义主要的物理链。**附加群组**可添加多达 7 个群组，每个群组都拥有自己的骨骼选择。非主群组可以继承主群组的设置，或通过**覆盖物理**进行覆盖。

## 骨骼选择

每个群组都有一个**选择骨骼**选择器，用于选择根骨骼。**排序**用于组织骨骼以便进行侧向连接——*最短路径*、*环形*、*线性* 或 *不排序*。**闭合循环**连接每个层级的第一个和最后一个骨骼，形成无缝环形结构。**跳过前 X 个骨骼**将初始层级排除在物理模拟之外，使腰部区域保持稳定。

## 物理模式

**自动**遵循系统级的默认设置。**PhysX** 使用基于关节的刚体物理，并配有盒子、胶囊或球体碰撞体。**XPBD** 使用基于粒子的网格模拟，可实现更稳定、连续的变形。该模式决定显示哪个设置面板。

## PhysX 设置

在使用 PhysX 模式时可见。包含**物理属性**（质量、阻尼、摩擦、求解器迭代）、**父子关节**（摆动/扭转驱动）、**侧向关节**（相邻骨骼之间的线性/角度连接）和**碰撞体**参数（类型、半径、长度）的嵌套面板。**第一个碰撞体长度**通常设置得更短，以避免与身体碰撞体干扰。

## XPBD 设置

在使用 XPBD 模式时可见。通过 XPartMesh 系统配置基于粒子的网格模拟，包括旋转、扭转和侧向合规性值。

## 可视化

**可视化身体**渲染物理实体的碰撞体形状。**可视化关节**显示关节限制和驱动目标作为线框模型。

# 子组件

## 主群组

单个裙子物理群组的嵌套配置。**选择骨骼**选择根骨骼链。**排序**组织骨骼以进行侧向连接。**闭合循环**连接每个层级的第一个和最后一个骨骼。**跳过前 X 个骨骼**将初始层级排除在物理模拟之外。**物理模式**选择 PhysX 或 XPBD（仅限主群组）。**可视化身体/关节**显示调试几何体。包含嵌套的**PhysX 设置**（具有物理属性、父子关节、侧向关节和碰撞体子配置），或**XPBD 设置**粒子网格配置。

### XPBD 设置

为毛发、布料和其他悬垂部分配置基于粒子的链或网格模拟。**旋转合规性**控制每个关节弯曲的程度（值越高，越灵活）。**扭转合规性**控制绕骨骼轴的旋转。对于网格模式，**侧向合规性**在相邻链之间添加交叉连接。

**刚度衰减**（10 的幂次方尺度）乘以每个层级的合规性——高于 1 的值使链向末端逐渐松弛。**质量衰减**（2 的幂次方尺度）降低每个层级的质量。**粒子锚点**将关节定位到片段上。**约束阻尼**平滑求解器步骤之间的振荡。**惯性**增加对运动变化的抵抗。**粒子半径**以毫米为单位设置碰撞体大小。**使用球形形状**用球体替换胶囊粒子进行调试。
