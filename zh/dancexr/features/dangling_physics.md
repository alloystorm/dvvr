# 悬挂物理

对模型悬挂部分（如彩带、围巾、尾部和松散饰品）应用链式物理效果。其工作原理类似于毛发物理，但针对更重、更慢的运动进行了调整，降低了弹簧刚度，以实现更夸张的摆动效果。


## 骨骼选择

使用 **选择骨骼** 来选择悬挂部分的根骨骼——系统会自动遍历子骨骼来构建链。**跳过前 X 根骨骼** 会使根骨骼未连接，以提供更稳固的基础。**最大层级** 限制了链的深度（0 = 无限制）。预设允许您在不同模型之间保存配置。


## 物理模式

**自动** 遵循系统默认设置。**PhysX** 使用关节式的刚体物理，配有胶囊体或球体碰撞器。**XPBD** 使用基于粒子的模拟，对于长链更稳定，并与场景碰撞层进行交互。该模式决定了显示哪个设置面板。


## PhysX 设置

在使用 PhysX 模式时可见。**弹簧力**（对数）控制刚度——比毛发物理更低的值会产生更宽松、更剧烈的摆动。**阻尼** 减少振荡。**刚度衰减** 柔化每个链级的弹簧，使尖端比基础更自由地摆动。**扭转限制** 限制骨骼轴周围的旋转。**限制力** 控制关节抵抗其约束的力度。默认情况下，**质量** 比毛发更高，以提供更多动量。**阻力** 添加空气阻力。**碰撞器半径** 设置碰撞球的大小；**碰撞器长度** 沿骨骼将其缩放为胶囊体。**锚点位置** 选择关节连接的位置（0 = 父骨骼，1 = 子骨骼）。


## XPBD 链

在使用 XPBD 模式时可见。使用 **旋转柔顺度**、**扭转柔顺度**、**刚度衰减**、**质量衰减** 和 **约束阻尼** 配置基于粒子的链模拟。**粒子锚点** 控制附着位置。碰撞层设置为 Cloth，以与场景的其余部分进行交互。


## 可视化

**可视化刚体** 渲染物理刚体的碰撞器形状。
**可视化关节** 显示关节限制和驱动目标作为线框小工具。


# 子组件

## XPBD

为毛发、布料和其他悬挂部分配置基于粒子的链或网格模拟。**旋转柔顺度** 控制链在每个关节处的弯曲程度（越高 = 越灵活）。**扭转柔顺度** 控制骨骼轴周围的旋转。
对于网格模式，**横向柔顺度** 在相邻链之间添加交叉连接。

**刚度衰减**（10 的幂尺度）在每个级别上乘以柔顺度——高于 1 的值使链逐渐变得更松弛，直到尖端。**质量衰减**（2 的幂尺度）降低每个级别上的质量。**粒子锚点** 在段上定位关节。**约束阻尼** 平滑求解步骤之间的振荡。**惯性** 为运动变化添加阻力。
**粒子半径** 设置碰撞器的尺寸（单位为毫米）。
**使用球体形状** 用球体替换胶囊粒子，用于调试。
