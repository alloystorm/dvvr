# 发光球体

生成附着在角色**骨骼**上的受物理驱动的发光球体。**形状**选择球体、菱形或水晶网格。**大小**缩放球体。切换附着点：**手部**、**髋部**和**躯干**。

**样式**面板控制外观，包含预设（发光、反射、水晶）、颜色、光泽、发光、金属度、强度、折射、阴影和拖尾。使用**舞台颜色**同步场景光照，或使用**随节拍闪烁**实现音频反应脉冲。

**物理**面板配置弹簧关节行为，包含预设（弹簧、悬挂、漂浮）、重力、碰撞、质量、距离、弹簧力、阻尼和速度限制。


# 子组件

## 样式

发光球体外观的嵌套配置。预设：**发光**（自发光）、**反射**（金属）、**水晶**（折射）。**使用舞台颜色**同步场景光照。**随节拍闪烁**将强度脉冲到音频。控制**颜色**、**光泽**、**发光**、**半径**、**金属度**、**强度**、**折射**、**投射阴影**和**拖尾**长度。

### 颜色

保存基本颜色和发光强度，用于音频反应元素。当启用自动更新时，发光会与颜色相乘，并随着节拍动画。

## 物理

发光球体物理行为的嵌套配置。预设：**弹簧**（无重力/碰撞）、**悬挂**（重力 + 碰撞）、**漂浮**（无重力/碰撞）。**重力**启用向下力。**碰撞**启用物理碰撞。**质量**、**距离**、**弹簧力**和**阻尼**配置弹簧关节。**最小/最大速度**限制速度。
