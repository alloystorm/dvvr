# 触手

投放并模拟附着在演员身上的程序化触手资源。每条触手都是由 XPBD 物理引擎驱动的、具有摆动、卷曲和吸引行为的分割网格。

**触手数量** 和 **长度** 设置数量和尺寸。**生成** 面板控制集群区域：**区域形状**（圆形或矩形）、**区域大小**、**宽高比** 和 **长度分布** 用于变化。**半径**、**收缩度** 和 **尖端半径** 塑造每条触手的厚度轮廓。

**行为** 面板驱动动画：**摆动速度**、**幅度** 和 **延迟** 创建有机摆动；**卷曲幅度** 和 **卷曲衰减** 添加朝尖端衰减的螺旋卷曲。**吸引力** 和 **吸引偏移** 将触手尖端拉向追踪的身体位置。**运动** 将动作同步到性动作系统；**运动幅度** 缩放响应。**缩回距离** 控制触手回缩的远度。

**材质** 和 **X光** 子面板控制外观和横截面渲染。**触手资源** 定义全局物理参数。更改数量或生成参数后，点击 **重建触手** 以重新生成网格。
