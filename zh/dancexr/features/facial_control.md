# 面部控制

为 XPS/FBX 模型提供了面部表情的直接滑块控制。
包括口型（A, I, U, E, O, Grinning, Smile 1-3, Frown）、眉毛姿势（Smile, Upper, Lower, Worry, Angry 1-2）和眼睑姿势（Blink, Jito, Wink, Laugh, Narrow），每个姿势都有其独立的滑块。

**使用口型同步** (Use Lip Sync) 根据当前播放的音频驱动口型变化。对于 XPS 模型，**禁用** (Disable) 会关闭所有面部动画，而 **范围** (Extent) 滑块则可以独立调整眼睑、嘴部和眉毛的运动幅度，以修正过度或动画不足的面部。
