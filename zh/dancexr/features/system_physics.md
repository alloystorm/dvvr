# 系统物理

描述场景的全局物理引擎选择、时间步长设置和重力。

## 默认引擎

**PMX 默认** 为带有内置刚体和关节（发型、裙子、配件）的 PMX 模型设置使用的物理引擎。请为使用 Unity 基于关节的物理，选择 *PhysX*；为使用粒子模拟，选择 *XPBD*；或选择 *关闭* 以禁用并释放这些骨骼给 XPS 定制物理工具使用。

**自定义物理默认** 为 XPS 工具（发型物理、布料物理、Softbody 等）设置的引擎。对于布料，XPBD 更稳定；PhysX 在某些硬件上可能更快。

## PhysX 设置

**重力** 设置应用于所有刚体的向下力。将其设置为零，可实现零重力舞台。**禁用碰撞** 可防止 XPS 粒子系统之间相互碰撞，当多个 Softbody 相互作用时，这可以减少抖动。

## XPBD 设置

保存 XPBD 引擎的模拟参数——有关详细信息，请查看 XPBD 设置子数据。

## 时间

**时间比例** 可独立于动画播放速度，减慢或加速物理模拟。较低的值使粒子和 Softbody 呈慢动作运动。**物理帧率** 将模拟锁定到一个固定时间步长：较低的值可降低 CPU 负载，较高的值以牺牲性能为代价提高稳定性。*灵活* 允许模拟以渲染帧率运行。
