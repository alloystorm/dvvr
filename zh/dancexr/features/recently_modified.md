# 最近修改的设置

这是一个快速访问列表，用于查看您最近更改的每个角色的设置。它可以让您无需通过角色菜单层级结构导航，即可重新打开相同的设置对话框。

可在角色菜单中找到——请参阅 [Actor menu & tools](actor_tools#actor-menu)。

---

## 存在的原因

角色菜单包含许多部分——材料、物理、服装、动作、行为等。当您正在反复修改某个单一功能时（例如调整头发物理、比较两种皮肤材质设置），直接跳回到最近使用的对话框比再次深入设置要快得多。

最近修改的设置为您提供了这种快捷方式。当您更改设置时，列表会自动更新；最近的更改将出现在顶部。

---

## 使用方法

<!-- TODO: confirm exact UI behavior. Below is what is implied by the actor_tools menu reference. -->

1. 打开角色菜单（点击角色的选择盘）。
2. 点击菜单顶部的 **最近修改**。
3. 列表中显示了您在此角色上调整的最新的设置，最新的设置位于顶部。
4. 点击任意条目即可跳转到相应的对话框。

此列表是针对每个角色的，当您重新加载模型或场景时，它会重置。
