# 缩放与偏移

调整角色在场景中的整体尺寸、位置和旋转。

**模型缩放** 使用 2 的幂指数缩放，因此微小变化也能产生精细的调整。**统一高度** 会将所有角色缩放到一致的平均人体高度，无论其原始模型比例如何——如果您想保留模型设计的尺寸，请禁用此功能。**地面偏移** 相对地面移动角色的高度。

**旋转** 围绕 Y 轴旋转角色。**X 偏移** 和 **Z 偏移** 水平移动位置。**吸附** 设置角色在场景中拖动时的增量大小——设置为零会完全禁用吸附功能。

预设包括 *统一真人尺寸*、*微缩*、*巨型* 和 *原始*。
