# 物理

配置包含内置刚体和关节定义（毛发、裙子、配饰）的 PMX 模型物理。

**物理模式** 用于选择引擎：*自动* 使用系统默认值；*PhysX* 运行 Unity 的基于关节的物理；*XPBD* 使用基于粒子的模拟；*关闭* 完全禁用 PMX 物理，并将其暴露为 XPS 物理工具（毛发物理、衣物物理等）的替代。

**可视化骨骼** 和 **可视化关节** 将物理形状和关节约束渲染为线框式小机关（gizmos）。嵌套的 **粒子属性** 面板定义全局 XPBD 设置，例如重力、刚度和阻尼。

**线性运动** 和 **角运动** 调整 PhysX 关节驱动响应——即物体跟随其动画目标时的积极程度。**选项** 暴露高级 PhysX 参数，例如求解器迭代次数和子步计数。**自动重置阈值** 在骨骼速度超过限制时将其恢复到动画位置，防止物理失控。

每个 **组** 部分对应于 PMX 文件中定义的物理关节组，包含每个组的启用/禁用、重力和碰撞设置。

**XPS 工具** 部分提供快速链接，链接到当 PMX 物理关闭时可用的替代物理系统。
