# 音频可视化

将音频频谱数据可视化为地面平面的光环覆盖层，实时响应节拍。


## 布局

可选择 *水平*、*垂直* 或 *圆形* 布局。*圆形* 将可视化环绕舞者，最适合圆形舞台。在 *水平* 或 *垂直* 模式下，可视化沿着地板的一个方向延伸。


## 光环设置

**大小** 缩放整体可视化区域。**深度** 控制可视化从中心延伸的远近。**光环大小** 调整每个光环的厚度。**数据缩放** 乘数音频振幅——放大微弱信号或减弱强信号。**颜色过渡** 控制光环之间颜色的平滑混合程度；值越低，形成的带状区域越清晰、越独立。

**形状变化** 随时间扭曲光环形状以创建有机运动，而 **间隙** 设置相邻光环之间的间隔。**对齐** 使光环相对于起始点进行偏移——这对于微调与舞台几何结构对齐非常有用。


## 光环颜色

光环颜色使用基础颜色和一个 *辉光* 强度，该辉光可以随着节拍脉动。预设包括 *动画色相*（缓慢循环通过频谱）和 *音乐同步辉光*（与音乐同步的辉光，适用于强调强拍）。


## 背景

可选的背景层位于光环后方，具有自己的颜色。可以应用纹理图像；*背景振动* 使其随着音频闪烁，产生反应性纹理效果。只有在禁用 *透明度* 时，背景才可见。


## 前景

前景层渲染在光环上方，具有独立的颜色和纹理控制。*前景缩放* 放大前景纹理；*前景振动* 添加音频反应闪烁。前景的颜色会乘以光环颜色，因此从一个明亮的背景开始，然后通过光环颜色进行着色。前景始终可见，并与下方的光环融合。


## 节拍时钟

当启用 *节拍时钟* 时，光环与检测到的 BPM 同步脉动，而非原始音频振幅——在节奏稳定的曲目中产生更清晰、更有韵律感的动画。*自动 BPM* 将从配置的 BPM 切换到实时检测到的节奏。


## 透明度

启用 *透明度* 会移除背景填充，只保留光环及其辉光，营造出适合暗色舞台的极简外观。阴影效果无论设置如何始终保持激活。


# 子组件

## 音频可视化

包含光环可视化布局、颜色、纹理和音频反应设置。

### 光环颜色

包含用于音频反应元素的基色和辉光强度。
当启用自动更新时，辉光会与颜色相乘并随着节拍动画。

### 背景颜色

包含用于音频反应元素的基色和辉光强度。
当启用自动更新时，辉光会与颜色相乘并随着节拍动画。

### 前景颜色

包含用于音频反应元素的基色和辉光强度。
当启用自动更新时，辉光会与颜色相乘并随着节拍动画。
