## 材料如何组织<a id="material-category"></a>

一个模型可能有数百种不同的材料。为了更有效地管理它们，我们按照以下方式组织材料：

* 纹理组
  为了使材料列表更易管理，我们按照它们使用的纹理文件对它们进行分组。大多数情况下，皮肤、服装、配饰和头发都使用不同的纹理贴图。因此，按照纹理进行分组为不同目的提供了良好的区分。

* 材料类别
  除了纹理分组外，我们还尝试通过检查材料和纹理名称来确定每种材料的目的。根据这些目的，我们创建了类别，以便更轻松地管理材料。
  这些类别包括：
  * 皮肤：身体、脸部、手臂和腿部属于这个类别，它们会自动分配皮肤着色器以获得最佳外观。
  * 头发：头发材料也有自己的着色器以获得最佳外观。
  * 眼睛：眼睛具有不同的默认值，如更高的光泽。
  * 嘴唇：嘴唇也分配了与标准材料不同的值。
  * 不透明：不属于前述类别的不透明材料。
  * 透明：不属于其他类别的透明材料。
  * 自定义：默认为空，但您可以将材料分配到此类别并对其进行自定义调整。

## 进行调整

### 材料列表<a id="material-list"></a>

您可以从演员菜单中找到材料列表。只需转到“材料”/“材料列表”，您将看到所有材料按其纹理组列出。

您可以在材料列表中进行的调整：
* 组特定控件：
  * 切换：控制整个组的可见性
  * 覆盖：如果您希望所有材料中的更改都适用于整个组，请打开“覆盖”并在其中进行调整。
  * 共享材料：打开此选项以强制整个组使用来自第一项的相同材料。这可能通过减少多个材料的开销来提高性能。请注意，有时个别材料具有不同的设置，此时使用共享材料可能会产生意外结果。
* 个别材料控制：
  * 更改类别：更改默认类别
  * 透明度：在自动、不透明和透明之间进行更改
  * 透明预处理：这是一种解决Alpha排序问题的技术。但它的缺点是只允许第一层显示。如果不需要，可以关闭它。
  * 透明预处理阈值：高于阈值的Alpha值将不使用预处理。
  * 表面：调整颜色、光滑度、金属和发光
  * 阴影：选择是否投下阴影
  * 阴影阈值：只有Alpha值高于阈值时才会投下阴影。

## 纹理

材料可以携带无限数量的纹理（自2025年9月起），每种纹理在表面外观的呈现中都扮演着不同的角色。材料设置中的纹理部分会列出所选材料上的每种纹理；点击某张纹理可以检查它，并重新分配其用途，如果DanceXR的自动检测出错。

DanceXR根据纹理文件名自动分配纹理类型和通道映射。大多数情况下您不需要更改任何东西——本节的其余部分是为您需要进行修改的情况准备的。

### 基本纹理类型

用于映射到材料单个角色的纹理：

| 类型 | 功能描述 |
|---|---|
| Color | 表面的主颜色（也称为漫反射/基础色/基础颜色）。 |
| Bump | 通过改变表面法线来增加表面起伏感——如皱纹、接缝、织物编织。 |
| Specular | 在旧版反射工作流程中定义高光颜色和锐度。DanceXR 使用 PBR 工作流程，因此反射数据被重定向到金属度和粗糙度贴图中。 |
| Emissive | 使表面看起来像发光。用于发光效果、屏幕、眼睛等。 |
| Alpha (不透明度) | 控制材料的透明度。 |

### 高级纹理类型

某些纹理将多个角色打包到一个图像的通道（R、G、B、A）中。DanceXR 支持两种：

- **Detail（细节）** — 一种平铺纹理，在基础颜色上添加精细的细节（皮肤毛孔、织物纹理）。
- **Mask（遮罩）** — 定义表面哪些区域受哪些属性影响（金属度、粗糙度、遮挡度、细节）。

对于这些纹理，您需要将纹理的每个通道分配给特定的用途。可用的通道用途包括：

| 用途 | 应用于 | 控制内容 |
|---|---|---|
| Metallic（金属度） | Mask贴图 | 表面在每个像素上的金属度。 |
| Occlusion（遮挡度） | Mask贴图 | 环境遮挡——缝隙中的柔和阴影。 |
| Smoothness（平滑度） | Mask贴图 | 表面有多平滑（像镜子一样）。 |
| Roughness（粗糙度） | Mask贴图 | 平滑度的反向——为在粗糙度工作流程而不是平滑度工作流程中创建的纹理提供。 |
| Bump X | Detail贴图 | 细节法线贴图的 X 轴。 |
| Bump Y | Detail贴图 | 细节法线贴图的 Y 轴。 |
| Detail Mask | Mask贴图 | 限制细节贴图应用的范围。 |

因此，一张单个遮罩纹理的红色通道可能驱动金属度，绿色通道驱动遮挡度，蓝色通道驱动平滑度——DanceXR 将每个通道单独暴露，您可以单独重新指向它们。

### 检查纹理

在纹理子菜单中：

- 纹理本身显示在面板顶部。
- 点击纹理可将其分解为单个通道（R、G、B、A），这样您就可以看到每个通道承载了什么信息——当您想知道一个打包遮罩纹理是如何制作的，这个功能特别有用。
- 在预览下方，选项允许您更改纹理的类型分配和通道用途。

这是确定一个陌生纹理用途最快的方法：打开它，查看通道，如有必要则重新分配。
