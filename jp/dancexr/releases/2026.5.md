# Release 2026.5

## DanceXR Operator の導入
DanceXR Operator は、DanceXR 専用のローカル AI バックエンドです。ゲームの横でローカルサーバーとして動作し、音声合成と大規模言語モデルによるチャットを統一された API の背後にまとめることで、別のクラウドサービスに依存せずに AI 会話、ロールプレイ、キャラクターボイス機能を DanceXR に提供します。

実際には、今回のリリースで新しくなった AI チャット基盤を支えているのが Operator です。自分のゲーミング PC 上でモデル管理と音声パイプラインをローカルに処理し、より長く、よりシーンに根ざしたキャラクターインタラクションを支える安定した土台になっています。

[Download DanceXR Operator](../features/operator)


## AIチャットの刷新
AIチャットは、キャラクター行動シミュレーションのためのエージェント型ワークフローを中心に再構築されました。Operator を専用のローカルバックエンドとして使うことで、別のサーバーマシンやクラウド接続なしでも一般的なゲーミング環境で動作させやすくなっています。

- **環境認識**: 新しいチャットシステムはシーン情報をより早い段階で読み込み、より一貫して利用します。これにより、キャラクターは現在の場所、時間帯、照明、ステージ構成などを踏まえて会話しやすくなりました。
- **記憶と継続性**: キャラクターは最近のやり取り、関係性、重要な出来事をより安定して保持できるようになりました。複数ターン履歴、意図追跡、セッション永続化、メモリ圧縮が組み合わさることで、同じ話の繰り返しに戻りにくくなっています。
- **インタラクションの流れ改善**: プロンプト処理、話者ターン、起動時の挙動、状態表示が見直され、長い会話でも破綻しにくくなりました。空メッセージ処理や Operator 起動の安定性も改善されています。
- **高品質TTS**: 更新された TTS パイプラインにより、音声品質が向上し、対応言語も広がりました。リアルタイム性は保ちつつ、音声が使えない場合のフォールバックや非音声言語への対応も改善されています。

## セックスモーションの洗練
今回のリリースでは、ペア型のプロシージャルセックスモーションもさらに洗練されました。重点は、動きをよりつながって見せ、読み取りやすくし、調整しやすくすることにあります。ここでの改善は、Sex Motion 3 の機能ページで説明されている、女性側が接触フレームを生成し男性側がそこへバインドするモデル主導の接触システムを土台にしています。

- **より自然なリアクションモーション**
  リアクション挙動は、スラストの伸びと体の動きにより素直に反応するようになりました。胴体の揺れ、脈動、減衰、オーバーシュート制御が洗練され、誇張しすぎず反応的に見える動きを作りやすくなっています。
- **オーガズムシミュレーション**
  興奮、ピークまでの buildup、収束の挙動は、より明確なモデル主導ロジックとして整理されました。刺激ベースで進めたい場合と、意図したリズムで制御したい場合の両方で扱いやすくなり、震えや表情強度のレイヤーもまとめて調整しやすくなっています。
- **適切な位置合わせと接触解決**
  接触計算は、接触フレーム、腰アンカー位置合わせ、カスタム上方向ベクトル、ローカル回転制御、より強い体回転連動に対応しました。これにより、ポーズが複雑になってもペアの位置関係を保ちやすくなっています。
- **モーションパラメータの調整性向上**
  速度、刺激、進行、減衰、クイックモーション設定まわりが整理されました。テンポや強弱の作り分けがしやすくなり、設定同士の見えにくい干渉に悩まされにくくなっています。
- **ポーズ、視線、表情制御の改善**
  ランダムポーズ更新、体の曲げ、ねじり、頭回転、強制ルックアット、フレーム移動などにより、シーン調整の自由度が上がりました。視線固定、下向き視線、まばたき、まぶた制御、表情強度の連動も改善され、機械的すぎない表現がしやすくなっています。

## バグ修正
- シーン保存ロジックを見直し、シーン更新フローも改善しました。
- Operator 起動と起動時安定性に関する複数の問題を修正しました。
- 新しいチャットワークフローにおける空メッセージ処理、エンドポイント名、プラットフォーム選択、状態・エラー表示を改善しました。
- メニューのインデントやモーションアイコン処理などの細部を整理しました。
- コードベース全体で小規模な null 安全性修正とクリーンアップを適用しました。
