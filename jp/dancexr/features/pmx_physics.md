# 物理

内蔵の剛体やジョイント定義（髪、スカート、アクセサリー）を含むPMXモデルの物理を設定します。

**物理モード**でエンジンを選択します：*Auto*はシステムのデフォルトを使用します；*PhysX*はUnityのジョイントベースの物理を実行します；*XPBD*はパーティクルベースのシミュレーションを使用します；*Off*はPMX物理を完全に無効にし、代わりにXPSの物理ツール（ヘア物理、クローシング物理など）を公開します。

**ボディの視覚化**と**ジョイントの視覚化**は、物理形状とジョイント制約をワイヤーフレームのギズモとしてレンダリングします。
ネストされた**パーティクルプロパティ**パネルは、重力、剛性、減衰などのグローバルなXPBD設定を定義します。

**線形運動**と**角運動**は、PhysXジョイントドライブの応答を調整します。これは、ボディがアニメーション化されたターゲットにどれだけ積極的に追従するかを決定します。**オプション**は、ソルバーイテレーションやサブステップカウントなどの高度なPhysXパラメータを公開します。
**自動リセットの閾値**は、速度が制限を超えた場合にボーンをアニメーション化された位置にスナップバックさせ、物理が制御不能に回転するのを防ぎます。

各**グループ**セクションは、PMXファイルで定義された物理ジョイントグループに対応し、グループごとの有効/無効、重力、衝突設定を備えています。

**XPSツール**セクションは、PMX物理をオフにしたときに利用可能な代替物理システムへのクイックリンクを提供します。
