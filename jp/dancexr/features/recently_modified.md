# 最近変更した設定

アクターごとに、最も最近変更した設定を一覧表示するクイックアクセスリストです。アクターメニューの階層を辿り直すことなく、同じダイアログを再開できます。

アクターメニューから利用可能 — [アクターメニュー＆ツール](/dancexr/features/actor_tools#actor-menu) を参照。

---

## 存在理由

アクターメニューには、マテリアル、物理、ドレッシング、モーション、動作など、多くのセクションがあります。単一の機能（毛髪の物理の調整、2つの肌マテリアル設定の比較）を反復的に調整している場合、再度深く掘り下がるよりも、最も最近使用したダイアログに直接ジャンプする方が速いです。

「最近変更した設定」は、そのショートカットを提供します。このリストは設定が変更されるたびに自動的に更新され、最も新しい変更が一番上に表示されます。

---

## 使用方法

<!-- TODO: confirm exact UI behavior. Below is what is implied by the actor_tools menu reference. -->

1. アクターメニューを開きます（アクターの選択ディスクをクリック）。
2. メニューの上部にある**最近変更した設定**をクリックします。
3. リストには、このアクターに対して最も最近調整した設定が、新しいものが一番上に表示されます。
4. エントリーをクリックすると、そのダイアログにジャンプします。

このリストはアクターごとに適用され、モデルまたはシーンをリロードするとリセットされます。
