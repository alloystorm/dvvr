# ライトボール

アクターのボーンに取り付けられる、物理演算駆動の光るオーブを生成します。**シェイプ**でボール、ダイヤモンド、またはクリスタルメッシュを選択します。**サイズ**でオーブの大きさをスケーリングします。アタッチメントポイントを切り替えることができます：**手**、**股関節**、および**胴体**。

**スタイル**パネルでは、プリセット（Glow、Reflective、Crystal）で外観を制御し、色、光沢、グロー、メタリック、強度、屈折、影、および軌跡を設定します。**ステージカラー**を使用してシーン照明と同期させるか、**ビートで点滅**を使用してオーディオ反応性のパルスを設定します。

**物理**パネルでは、プリセット（Spring、Hanging、Floating）を使用してスプリングジョイントの挙動を設定し、重力、衝突、質量、距離、スプリングフォース、減衰、および速度制限を構成します。

# サブコンポーネント

## スタイル

ライトボールの外観に関するネストされた設定です。プリセット：**Glow**（エミッシブ）、**Reflective**（メタリック）、**Crystal**（屈折）。**ステージカラー**はシーン照明と同期します。**ビートで点滅**はオーディオに強度をパルスさせます。**色**、**光沢**、**グロー**、**半径**、**メタリック**、**強度**、**屈折**、**影のキャスト**、および**軌跡**の長さを制御します。

## 物理

ライトボールの物理的な挙動に関するネストされた設定です。プリセット：**Spring**（重力/衝突なし）、**Hanging**（重力＋衝突）、**Floating**（重力/衝突なし）。**重力**を有効にすると下向きの力が加わります。**衝突**を有効にすると物理的な衝突が可能になります。**質量**、**距離**、**スプリングフォース**、および**減衰**でスプリングジョイントを設定します。**最小/最大速度**で速度をクランプします。
