# 揺れ乳房オーバーレイ

音楽のビートに同期して駆動する、リズミカルな胸の揺れモーションを追加します。
**Intensity** は振幅を、**Speed** はビート周波数を調整します（2のべき乗）。**Motion Pattern** は、水平のみ (−1)、垂直のみ (1)、または複合 (0) の揺れ方向の間でブレンドされます。

**Time Pattern** でリズムを選択します：*Constant* はビートごとに振動します；*Pulse* は2ビートごとにオン/オフを繰り返します；*Gradual* は、フレーズを通して強度を増していく、事前定義された16ビートパターンに従います。**Abs Mode** は、中立位置で跳ね返り、より誇張された効果を生み出します。
