# 遠隔アクセス

リモートコントロールを使用すると、Androidアプリを、同じローカルネットワーク上の別のデバイス（PCやQuestなど）で実行されているDanceXRのワイヤレスコントローラーとして機能させることができます。メインのセッションが大きな画面で再生されている間、スマートフォンまたはタブレットからシーン、モーション、および設定を変更できます。

## 要件

* DanceXR AndroidアプリがインストールされたAndroidデバイス。
* 同じローカルエリアネットワーク（LAN）上で動作するPC、Quest、またはその他のDanceXRホスト。
* ネットワークがローカルのデバイス検出を許可していること（クライアント分離がないこと）。

## 接続方法

1. ホストデバイスで**遠隔アクセス**を有効にします。
2. **ポート番号**と安全な**パスワード**を設定します。
3. Androidアプリを開き、メインメニューから**リモートコントロール**を選択し、検出されたデバイスからホストを選択します。
4. メニューとタッチパッドの制御を同期するためにプロンプトが表示されたら、パスワードを入力します。
