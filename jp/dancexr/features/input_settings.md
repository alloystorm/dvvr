# コントローラー入力

入力設定では、キーボード/マウスとゲームコントローラー（VRハンドコントローラーを含む）の両方に対して、コントローラー入力をマッピングおよびキャリブレーションできます。


## オプション

**トラックパッドモード**は、コントローラーのタッチパッドの動作を変更します。有効にすると、軸の値はパッドボタンが物理的に押された場合にのみ出力されます。

**二次軸を使用**を有効にすると、ジョイスティックとトラックパッドの両方を持つコントローラー（WMRコントローラーなど）からの入力を読み取ることができます。

**ゲームパッドを使用**を有効にすると、VRモードで実行している場合でもゲームコントローラー/ゲームパッドを使用できます。

**グリップ入力を使用**は、グリップボタンの入力をオン/オフします。これをオフにすると、誤って握ったり、グリップによるドラッグ操作が発生するのを防げます。

**グリップしきい値** (0.0～1.0) は、グリップボタンのアクチュエーションしきい値を設定します。意図しない作動を防ぐため、非常に感度の高いコントローラー（Valve Indexなど）に有用です。


## コントローラーマッピング

コントローラー上の様々な軸とボタンのマッピングをカスタマイズできます：
* **軸**は、カメラの移動、回転、タイムラインスクラビング、またはフレームステップなどのアクションにマッピングできます。
* **ボタン**は、動作の再生、次のモデル/前のモデルの読み込み、スローモーション、または物理演算のリセットなどのアクションにマッピングできます。
* **Altキー**を割り当てて、キーボードキーをコントローラーボタンにマッピングすることができます。
