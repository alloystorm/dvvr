# 自動リセット

速度が安全な制限を超えた場合、ボーンとその子ボーンを自動的にリセットします。極度の力やテレポートによる物理エンジンの爆発やメッシュの変形を防ぎます。


## しきい値

物理がリセットされる速度のしきい値を設定します。低い値はリセットをより積極的にトリガーするため、クリッピングを防ぐことができますが、高速移動中に目に見えるカクつきを引き起こす可能性があります。高い値は、介入する前に、より極端な動きを許容します。
