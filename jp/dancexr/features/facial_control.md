# 顔の制御

XPS/FBXモデルの顔の表情に直接的なスライダーを提供します。
口の形状（A、I、U、E、O、ニヤリ、笑顔1〜3、不機嫌）、眉のポーズ（笑顔、上向き、下向き、心配、怒り1〜2）、およびまぶたのポーズ（まばたき、Jito、ウィンク、笑い、細め）それぞれに専用のスライダーがあります。

**リップシンクを使用**が現在再生中の音声から口の形状を制御します。XPSモデルの場合、**無効**にするとすべての顔のアニメーションがオフになり、**範囲**スライダーでまぶた、口、眉の動きを個別にスケール調整し、過剰または不十分なアニメーションの顔を修正できます。
