# オブジェクトの分離

選択したボーンをキャラクターモデルから分離し、独立した物理シミュレーションを適用します。これにより、アクセサリーや持っている物体が動的に脱落するようになります。

## ボーンの選択

**ボーンの選択**を使用して、どのボーンを物理オブジェクトとして分離するか指定します。選択できるのは、非キネマティックなボーンのみです。

## 物理演算

**重力**をトグルすることで、分離されたボーンに重力の影響を与えます。**質量**は剛体（rigidbody）の重さを制御します。重い物体ほど速く落ち、より強く押し当たります。**減衰**は空気抵抗を追加し、時間の経過とともに動きを緩めます。

## コライダー

衝突形状を選択します。*なし*、*球体*、または *カプセル* のいずれかです。**コライダー半径**は、球体／カプセルの幅を設定します。**コライダー長さ**は、カプセルをボーン軸に沿って延長します。
