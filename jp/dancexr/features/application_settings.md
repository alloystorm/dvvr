# アプリケーション設定

スタートアップ動作とコンテンツ処理に関するグローバルなアプリケーション設定。

## スタートアップ

**自動ロード**は、アプリケーション起動時に何が起こるかを制御します。
*None* は空のステージで開始し、*Last Actor* は最後に使用されたアクターモデルをロードし、*Last Scene* は最後に保存されたシーンを復元します。

## テクスチャ読み込み

**Flip DDS Compressed** および **Flip DDS Uncompressed** は、DDSテクスチャがロード時に垂直方向に反転するかどうかを制御します。テクスチャが上下逆さまに見える場合はこれらを有効にしてください。どの設定が必要かはテクスチャの圧縮形式に依存します。

## UI動作

**Gizmo 3rd Axis** は、トランスフォームグイズモ使用時のマウスホイールの動作を変更します。*Rotation* はオブジェクトを選択された軸に沿って回転させ、*Depth* はオブジェクトを近づけたり遠ざけたりします。

**ローカライズされた名前を使用** は、オリジナルファイル名ではなく、利用可能な場合はローカライズされた言語でのコンテンツ名を表示します。

**保存からタグを復元** は、コンテンツキャッシュが再構築されたか破損した場合に、保存された設定からタグデータを復元しようとします。
