**ShiftPiano** is an open-source web-based piano app with high-quality acoustic grand piano soundfonts. It runs directly in any modern browser — no installation required.

[Try it now](https://alloystorm.github.io/shiftpiano/){: .btn .btn--primary target="_blank" rel="noopener"} [View on GitHub](https://github.com/alloystorm/shiftpiano){: .btn target="_blank" rel="noopener"}

---

## Features

- **Acoustic grand piano sound** — high-quality MP3 soundfonts for realistic piano playback
- **Runs in the browser** — no installation, no plugins, works on any modern browser
- **Open source** — MIT licensed, free to use and modify
