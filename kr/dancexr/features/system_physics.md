# 시스템 물리

장면의 글로벌 물리 엔진 선택, 시간 단계 설정, 중력 설정입니다.


## 기본 엔진

**PMX 기본**은 내장된 리지드바디와 본(머리카락, 치마, 액세서리 등)를 사용하는 PMX 모델에 적용되는 물리 엔진을 설정합니다. Unity의 조인트 기반 물리에는 *PhysX*를, 입자 기반 시뮬레이션에는 *XPBD*를 선택하거나, 비활성화하고 그 본들을 XPS 사용자 지정 물리 도구에서 사용할 수 있도록 하려면 *끄기*를 선택하십시오.

**사용자 지정 물리 기본**은 XPS 도구(머리카락 물리, 천 물리, Softbody 등)용 엔진을 설정합니다. XPBD는 천에 더 안정적이며, PhysX는 일부 하드웨어에서 더 빠를 수 있습니다.


## PhysX 설정

**중력**은 모든 리지드바디에 적용되는 하강 힘을 설정합니다. 무중력 무대를 위해 0으로 설정하십시오. **충돌 비활성화**는 XPS 입자 시스템끼리의 충돌을 방지하여, 여러 Softbody가 상호 작용할 때 발생할 수 있는 떨림을 줄여줍니다.


## XPBD 설정

XPBD 엔진의 시뮬레이션 매개변수를 담고 있습니다. 자세한 내용은 XPBD 설정 하위 데이터를 참조하십시오.


## 시간

**시간 스케일**은 애니메이션 재생 속도와 독립적으로 물리 시뮬레이션을 느리게 하거나 빠르게 합니다. 값이 낮을수록 입자와 Softbody가 슬로우 모션으로 움직입니다. **물리 FPS**는 시뮬레이션을 고정된 시간 단계로 잠급니다. 낮은 값은 CPU 부하를 줄이고, 높은 값은 성능을 희생하는 대신 안정성을 향상시킵니다. *유연*은 렌더 프레임 속도에서 시뮬레이션이 실행되도록 합니다.


# 하위 구성 요소

## XPBD 설정

XPBD 솔버 설정을 담고 있습니다: 서브스텝, 반복 횟수, 충돌 간격, 자가 충돌, 바디 충돌, 지면 마찰, 확장 반경 및 병렬 해결.
