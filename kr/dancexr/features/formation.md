# 형성

형성 기능은 배우들이 장면에 배치되는 방식을 제어하며, 배우 위치를 재설정하는 편리한 도구를 제공합니다.

## 형성 유형

다양한 레이아웃 패턴 중에서 선택할 수 있습니다:

* **V 모양**: X Dist와 Y Dist를 모두 사용하여 배우들을 V 모양으로 배치합니다.
* **A 모양**: X Dist와 Y Dist를 모두 사용하여 배우들을 A 모양으로 배치합니다.
* **라인**: X Dist를 배우들 사이의 거리로 사용하여 수평하게 배열합니다.
* **원**: X Dist가 반지름을, Y Dist가 오프셋을 제어하는 원 위에 배우들을 배치합니다.
* **제로**: 모든 배우에 대해 0 오프셋과 회전을 설정합니다. 이는 개별 위치가 이미 동작으로 구현된 다중 출연자 댄스에 유용합니다.
* **팬 모양**: 배우들을 부채꼴 레이아웃으로 배열합니다.

## 설정

**X Dist**와 **Y Dist**는 선택된 형성의 간격 스케일을 제어합니다.

**회전 (Rotation)**: 각 개별 배우를 회전시킵니다.

**방향 (Orientation)**: 전체 형성 그룹을 통째로 회전시킵니다.

**오프셋 X**와 **오프셋 Y**: 형성의 중앙 위치를 이동시킵니다.

## 관객

관객으로 레이블이 지정된 배우들은 주 무대를 중심으로 원을 그리는 별도의 형성 그룹에 배치됩니다.

* **관객 형성 사용**: 관객 위치를 활성화합니다.
* **관객 간격 (Spectator Dist)**: 관객의 반지름을 설정합니다.
* **관객 회전**: 관객들이 무대 중앙을 바라보도록 회전시킵니다.
