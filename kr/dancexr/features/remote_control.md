# 원격 제어

원격 제어 기능을 사용하면 **안드로이드 앱**이 동일한 로컬 네트워크에 있는 다른 장치(예: PC 또는 Quest)에서 실행되는 DanceXR의 무선 컨트롤러 역할을 할 수 있습니다. 메인 세션이 대형 화면에서 재생되는 동안, 휴대폰이나 태블릿에서 장면, 모션 및 설정을 변경할 수 있습니다.

## 요구 사항

* DanceXR 안드로이드 앱이 설치된 Android 장치.
* 동일한 로컬 영역 네트워크(LAN)에서 실행되는 PC, Quest 또는 기타 DanceXR 호스트.
* 네트워크는 로컬 기기 검색을 허용해야 합니다 (클라이언트 격리 불가).

## 연결 방법

1. 호스트 장치에서 **원격 제어**를 활성화합니다.
2. **포트 번호**와 안전한 **비밀번호**를 설정합니다.
3. 안드로이드 앱을 열고 메인 메뉴에서 **원격 제어**를 선택한 후, 검색된 기기 목록에서 호스트를 선택합니다.
4. 메뉴 및 터치패드 컨트롤을 동기화하려면 요청 시 비밀번호를 입력합니다.
