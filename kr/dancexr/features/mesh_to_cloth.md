# 메쉬를 천으로

액터에 있는 모든 렌더 메쉬를 입자 기반 천 시뮬레이션으로 변환합니다. 각 메쉬는 고정 정점(천을 고정하는 본) 및 시뮬레이션 매개변수를 선택하는 전용 설정 패널을 갖습니다.

**Gradual Enable**은 설정된 시간 동안 시뮬레이션을 점진적으로 활성화하여 메쉬가 갑작스럽게 제자리로 '튀는' 현상을 방지합니다. 중첩된 **Particle Props** 패널은 모든 변환된 메쉬에 적용되는 강성, 감쇠, 중력 배율과 같은 전역 물리 속성을 정의합니다.
