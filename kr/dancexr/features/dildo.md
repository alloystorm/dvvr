# 딜도

<!-- TODO: confirm exact UI labels and configuration. Drafted from related Sex Overlay docs and the prop-attachment system. -->

액터에게 부착되는 딜도 프로프를 설정합니다. 단일 액터와 독립적으로 사용하거나, 남성 캐릭터가 로드되지 않은 경우 페어링된 동작의 남성 측면으로 사용됩니다.

---

## 설정

1. 관련 액터에 딜도를 활성화합니다.
2. 부착 본/위치를 선택합니다. 일반적으로 골반(pelvis)에 고정됩니다.
3. 크기와 오프셋을 조정합니다.
4. 선택 사항: [섹스 동작](sex_motion_3) 또는 [자동 업데이트](autoupdate)에 연결된 애니메이션 동작(추진 패턴)을 구성합니다.

상세한 부착 구성은 [섹스 오버레이 및 딜도 설정](smo_config)에서 확인하실 수 있습니다.

## 동작과의 페어링

- [섹스 동작 3](sex_motion_3)와 함께 사용 시 — 딜도가 페어링된 동작에서 남성 역할을 수행할 수 있습니다. 동작의 접촉 프레임 시스템이 딜도를 여성 접촉 지점에 맞춥니다.
- [자동 업데이트](autoupdate)와 함께 사용 시 — 오디오 레벨이나 비트에 맞춰 추진 패턴을 구동하여 음악에 동기화된 동작을 구현합니다.
