# 모션 설정

액터 모델에 모션 데이터(VMD, BVH 등)를 적용하는 방식(스케일링, 타이밍 오프셋, IK 처리, 포즈 수정 등)을 제어합니다.

## 모션 매개변수

**모션 스케일(Motion Scale)**은 모션 파일의 모든 위치 이동을 곱하여 적용합니다. 애니메이션이 모델의 비율에 비해 과장되거나 너무 미묘할 때 유용합니다. **수직 위치 스케일(Vertical Pos Scale)**은 본 중심의 상/하 움직임을 개별적으로 스케일링하여 바운싱이나 점프하는 동작을 줄입니다.

**미러(Mirror)**는 왼쪽과 오른쪽 모션 데이터를 뒤집습니다. 주로 반대편 우세(opposite-side dominance)를 위해 제작된 VMD 파일에 유용합니다. **시간 오프셋 백분율(Time Offset Percent)**과 **시간 오프셋 초(Time Offset Seconds)**는 액터가 애니메이션 타임라인에서 재생을 시작하는 시점을 조정합니다. 여러 액터를 동기화하거나 모션의 시작 지연을 보상할 때 유용합니다.

**다리 각도(Leg Angle)**는 다리 본에 작은 회전 편향을 추가하여 애니메이션 전체에 걸쳐 스탠스 폭이나 발끝 벌림 각도를 조정합니다.

## IK 설정

**본 상속(Inherit Bones)**은 PMX 파일에 정의된 부모 상속 제약(예: 팔 본을 따라 움직이는 손 본)을 준수합니다. **모션 다리 IK(Motion Leg IK)**는 모션 처리 중 범용적인 2본 다리 IK 패스를 적용하여 전용 IK 본이 없는 모델에서도 발이 땅에 붙는 동작을 가능하게 합니다. **모델 IK(Model IK)**는 PMX 자체에 정의된 IK 링크 체인을 사용합니다. 모델 IK와 모션 IK가 충돌할 경우, 둘 중 하나를 비활성화해 보세요.

## 스펙테이터

액터를 스펙테이터로 지정합니다. 이 경우 해당 액터는 포메이션 패턴 및 조명 할당에서 제외되며, 청중이나 배경 캐릭터에게 유용합니다.

## 포즈 조정

어떤 모션이 적용되기 전에 기본 T 포즈 본 각도를 회전시킵니다. 개별 본 그룹별 회전 제어에 대해서는 중첩된 패널을 참조하세요. 이 기능은 임포트된 휴식 포즈가 애니메이션의 가정을 따르지 않는 모델을 수정하는 주요 방법입니다.

## 시각화

**가상 본 표시(Show Virtual Bones)**, **본 시각화(Visualize Bones)**, **IK 목표 시각화(Visualize IK Targets)**는 스켈레톤 계층 구조와 IK 목표 위치에 대한 디버그 기즈모를 렌더링합니다 (PMX 모델 전용).

# 하위 구성 요소

## sd_pose

액터의 모든 주요 본 그룹에 대해 기본(제로 포즈) 회전을 조정합니다. 이 값은 본을 원래 임포트된 T 포즈에서 회전시켜, 팔이 너무 넓게 있거나, 다리가 너무 가깝거나, 손가락이 잘못 펼쳐진 모델을 수정할 수 있도록 합니다.

각 설정은 각도(X/Y/Z)로 된 3축 회전값입니다. 왼쪽 본은 자동으로 오른쪽으로 미러링됩니다. **약지(Ring Finger)**와 **중지(Middle Finger)**는 각각 새끼손가락과 검지손가락에 대한 스칼라 승수로, 손가락 컬 비율을 유지합니다.
