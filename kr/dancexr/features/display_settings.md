## 개요
디스플레이 설정을 통해 UI, 화면 해상도 및 프레임 속도에 관한 구성을 변경할 수 있습니다.

## UI 설정
* **UI 크기**: UI의 크기를 변경합니다. 고해상도 화면에서 UI가 너무 작게 보일 때 유용합니다.
* **UI 너비**: 메뉴 UI의 너비를 조절합니다.
* **최대 UI 높이**: 메뉴 패널의 최대 높이를 조절합니다.
* **최대 메뉴 깊이**: 동시에 표시되는 최대 중첩 메뉴 깊이를 설정합니다.
* **들여쓰기**: 메뉴 내 들여쓰기 크기를 조절합니다.
* **메뉴 고정 사용**: 부모 메뉴가 시야 밖으로 이동하여 메뉴 뒤쪽 콘텐츠를 위한 공간을 만들 수 있습니다.
* **슬라이더 조작 시 UI 숨기기**: 슬라이더를 조절하는 동안 나머지 UI를 숨깁니다.
* **방향 입력으로 UI 조작**: 게임패드 방향키로 메뉴를 탐색할 수 있습니다.
* **로드 후 메뉴 재개**: 씬 로드 완료 후 마지막으로 열었던 메뉴를 복원합니다.
* **루트 메뉴 토글**: 메뉴가 이미 표시되어 있을 때 루트 메뉴 버튼으로 메뉴를 닫습니다.

## 화면 해상도
화면 해상도와 창 모드를 제어합니다. 전체 화면과 창 모드 간 전환이 가능합니다.

이 설정은 자동으로 적용되지 않습니다. **적용** 버튼을 클릭해야 변경 사항이 적용됩니다.

## 프레임 속도
목표 프레임 속도와 FPS 카운터 표시 위치를 제어합니다.

첫 번째 두 가지 프레임 속도 옵션은 특별합니다:
* **VSync**: 모니터의 재생 빈도에 동기화합니다.
* **최대**: 상한 없이 가능한 한 빠르게 렌더링합니다.

## VR 전용 설정
이 설정은 VR 모드에서만 사용할 수 있습니다.
* **UI 항상 위에 표시**: 다른 오브젝트에 가려져 있을 때도 UI가 항상 위에 렌더링되도록 합니다. 기본적으로 활성화되어 있습니다.
* **UI 자동 복귀**: 활성화 시 UI가 시야 밖으로 벗어나면 서서히 시야 안으로 돌아옵니다.
* **UI 거리**: UI 패널이 사용자 머리로부터 얼마나 떨어져 있는지 조절합니다.
* **VR에서 마우스 모드**: 핸드 컨트롤러 없이 마우스를 VR 포인터로 사용합니다.
* **마우스 감도**: VR 마우스 모드에서 포인터 감도를 조절합니다.

### 포인터
VR 상호작용에 사용되는 포인터 레이를 보정합니다.
* **방향**: 포인터 레이의 방향 각도를 조절합니다.
* **방위**: 포인터의 방위 오프셋을 조절합니다.
* **오프셋**: 포인터 원점의 위치 오프셋을 조절합니다.
* **포인터 업데이트**: 현재 포인터 보정 설정을 적용합니다.

## 포비에이티드 렌더링 (v2025.12)
포비에이티드 렌더링은 중앙을 선명하게 유지하면서 주변 영역을 낮은 해상도로 렌더링하여 GPU 부하를 줄입니다.
* OpenXR 호환 PC VR 헤드셋에서 지원됩니다.
* Quest 독립형 기기의 AR 및 VR 모드 모두에서 작동합니다.
* **레벨**: 시야 가장자리를 낮은 해상도로 렌더링하는 강도를 조절합니다 (0–1).
