## 개요
디스플레이 설정은 UI, 화면 해상도 및 프레임 속도에 대한 구성을 변경할 수 있습니다.

## UI 설정
* UI 크기: 이 설정은 UI의 크기를 변경할 수 있게 해줍니다. 고해상도 화면을 사용하는 경우 UI가 너무 작아 보일 때 유용합니다.
* UI 너비: 메뉴 UI의 너비를 조절합니다.
* 메뉴 고정 사용: 부모 메뉴가 보이지 않도록 이동하여 메뉴 뒤의 공간을 마련할 수 있게 해줍니다.

## 화면 해상도
이 설정은 화면의 해상도를 변경할 수 있게 해줍니다. 전체 화면과 창 모드 간 전환이 가능합니다.

이 설정은 자동으로 적용되지 않습니다. 변경 사항을 적용하려면 "적용" 버튼을 클릭해야 합니다.

## 프레임 속도
이 설정은 프로그램의 목표 프레임 속도를 변경하고 FPS가 표시되는 위치를 결정할 수 있게 해줍니다.

첫 번째 2개의 프레임 속도 옵션은 특별합니다.
* 첫 번째는 "VSync"로, 모니터의 새로고침 빈도를 목표로 합니다.
* 두 번째는 "Max"로, 가능한 한 빠르게 렌더링합니다.

## VR 특정 설정
이 설정은 VR 모드에서만 사용할 수 있습니다.
* UI 항상 위에 표시: UI가 다른 객체에 가려져 있을 때도 항상 위에 렌더링되도록 보장합니다. 기본적으로 활성화되어 있습니다.
* UI 자동 복귀: 이 설정이 활성화되면 UI가 보이지 않을 때 서서히 시야로 돌아옵니다.
* UI 거리: UI가 머리에서 얼마나 떨어져 있는지를 조절합니다.

## 포비에이티드 렌더링 (v2025.12)
* OpenXR 포비에이티드 렌더링 지원 추가.
* 지원되는 PC VR 헤드셋에서 작동합니다.
* AR 및 VR 모드 모두에서 퀘스트 독립형에서 작동합니다.
