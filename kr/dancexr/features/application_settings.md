## 시작

**자동 로드(Auto Load)**는 애플리케이션 시작 시 발생하는 동작을 제어합니다.
*없음(None)*은 빈 무대에서 시작하며, *최근 배우(Last Actor)*는 가장 최근에 사용된 배우 에셋을 로드하고, *최근 씬(Last Scene)*은 마지막으로 저장된 씬을 복원합니다.

## 텍스처 로딩

**DDS 압축 뒤집기(Flip DDS Compressed)**와 **DDS 압축 해제 뒤집기(Flip DDS Uncompressed)**는 DDS 텍스처가 로드 시 수직으로 뒤집히는지 여부를 제어합니다. 텍스처가 거꾸로 보이는 경우 이 옵션을 활성화해야 합니다. 어떤 설정을 사용해야 하는지는 텍스처 압축 형식에 따라 달라집니다.

## UI 동작

**그리즈모 3차 축(Gizmo 3rd Axis)**은 변형 그리즈모를 사용할 때 마우스 휠의 동작을 변경합니다. *회전(Rotation)*은 선택된 축을 따라 오브젝트를 회전시키고, *깊이(Depth)*는 오브젝트를 더 가깝거나 더 멀리 이동시킵니다.

**번역된 이름 사용(Use Translated Names)**은 원본 파일 이름 대신 사용 가능한 경우 현지화된 언어의 콘텐츠 이름을 표시합니다.

**저장된 태그 복구(Recover Tags From Save)**는 콘텐츠 캐시가 재구축되거나 손상된 경우 저장된 설정에서 태그 데이터를 복원하려고 시도합니다.
