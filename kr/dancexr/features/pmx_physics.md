# 물리

내장된 강체 및 관절 정의(머리카락, 치마, 액세서리 등)를 포함하는 PMX 모델의 물리 설정을 구성합니다.

**물리 모드**에서 엔진을 선택합니다: *자동*은 시스템 기본값을 사용하고; *PhysX*는 Unity의 관절 기반 물리를 실행하며; *XPBD*는 입자 기반 시뮬레이션을 사용하고; *해제*는 PMX 물리 자체를 비활성화하고 XPS 물리 도구(헤어 물리, 옷감 물리 등)를 대체하여 노출합니다.

**몸체 시각화**와 **관절 시각화**는 물리 형태와 관절 제약 조건을 와이어프레임 기즈모로 렌더링합니다. 중첩된 **입자 속성** 패널은 중력, 강성, 감쇠와 같은 전역 XPBD 설정을 정의합니다.

**선형 운동**과 **각 운동**은 PhysX 관절 구동 반응을 조정합니다. 즉, 몸체가 애니메이션 목표를 얼마나 적극적으로 따르는지를 결정합니다. **옵션**은 솔버 반복 횟수 및 서브스텝 횟수와 같은 고급 PhysX 매개변수를 노출합니다. **자동 재설정 임계값**은 속도가 제한을 초과하면 본을 애니메이션 위치로 되돌려 물리 현상이 통제 불능 상태가 되는 것을 방지합니다.

각 **그룹** 섹션은 PMX 파일에 정의된 물리 관절 그룹에 해당하며, 그룹별 활성화/비활성화, 중력 및 충돌 설정을 포함합니다.

**XPS 도구** 섹션은 PMX 물리 기능을 끄면 사용 가능한 대체 물리 시스템으로 가는 빠른 링크를 제공합니다.
