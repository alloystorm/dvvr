# 관전자 모드

관전자 모드는 에셋을 수동 관찰자로 지정합니다. 관전자 에셋은 씬에 로드된 상태로 유지되지만, "모든 에셋에" 작용하는 기능(자동 할당 동작, 포메이션, 그룹 댄스 생성 등)에서는 제외됩니다.

에셋 도구 메뉴에서 토글할 수 있습니다 — [에셋 메뉴 & 도구 → 도구 메뉴](/dancexr/features/actor_tools#tools-menu)를 참조하세요.

---

## 사용 시점

- 무대 위가 아닌 청중석에 서 있는 에셋이 필요할 때.
- 포메이션의 일부가 아닌 소품이나 배경으로 씬에 에셋을 배치하고 싶을 때.
- 한 에셋으로 설정을 테스트하면서 다른 에셋들은 로드 상태로 유지하고 싶을 때.

---

## 관전자 에셋으로 설정 시 변경되는 사항

- 새로운 댄스 세트가 로드될 때 **자동 할당 동작**에 의해 건너뜁니다.
- **포메이션** 배치에서 제외됩니다.
- 에셋 전반에 걸친 **자동 댄스** 생성에서 건너뜁니다.
- 선택 디스크와 에셋 메뉴는 여전히 사용 가능합니다 — 관전자 에셋은 여전히 수동으로 포즈를 취하거나, 옷을 입히거나, 움직일 수 있습니다.

---

## 관전자 상태 토글하기

1. 에셋의 선택 디스크를 클릭하여 에셋 메뉴를 엽니다.
2. 에셋 이름 옆의 렌치-망치 아이콘을 클릭하여 **도구 메뉴**를 엽니다.
3. **관전자**를 클릭하여 켜거나 끕니다.

이 설정은 에셋별로 적용됩니다. <!-- TODO: confirm whether it persists with scene save -->
