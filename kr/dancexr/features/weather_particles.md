## 개요
날씨 파티클은 비, 눈, 낙엽 등의 환경 조건을 시뮬레이션하는 대기 파티클 효과를 씬에 추가합니다. 내장 프리셋: **끄기**, **먼지**, **비**, **눈**, **꽃잎**, **낙엽**, **반짝이**.

## 파티클 설정

* **생성 속도** (0–2): 파티클의 밀도를 조절합니다. 값이 높을수록 파티클이 많아집니다.
* **중력**: 파티클이 떨어지는 강도를 조절합니다. 음수 값은 파티클이 아래로 떨어지게 합니다.
* **속도**: 생성 시 파티클의 초기 속도를 설정합니다.
* **난류**: 파티클에 무작위 움직임을 추가하여 더 자연스러운 외관을 만듭니다.
* **항력**: 파티클에 적용되는 공기 저항으로, 이동 중에 감속시킵니다.
* **튀기기**: 파티클이 지면이나 씬 오브젝트에 부딪혔을 때 튀어오르는 정도를 조절합니다.
* **회전**: 개별 파티클의 회전 속도입니다.
* **충돌**: 활성화하면 파티클이 씬 오브젝트를 통과하지 않고 충돌합니다.
* **위에서 생성**: 활성화하면 파티클이 생성 영역의 상단에서 생성됩니다. 비활성화하면 볼륨 전체에 무작위로 생성됩니다.
* **텍스처**: 사용할 파티클 텍스처 또는 유형을 선택합니다.
* **크기**: 각 파티클의 크기를 조절합니다.
* **수명**: 각 파티클이 사라지기 전에 표시되는 시간을 설정합니다.
* **높이**: 파티클 생성 영역의 높이를 설정합니다.
* **지면 머무름** (0+): 착지 후 파티클이 지면에 남아 있는 시간입니다. 0이면 접촉 시 즉시 사라집니다.

## 빗물 셰이더
빗물 셰이더는 카메라 렌즈에 빗방울이 맺히는 것을 시뮬레이션하는 스크린 스페이스 효과를 적용하여 비 날씨 중 몰입감을 높입니다.

## 오디오
빗소리나 환경 소음과 같은 날씨 오디오의 볼륨을 조절합니다.
