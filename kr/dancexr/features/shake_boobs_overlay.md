# 셰이크 브라 오버레이

음악 비트에 의해 구동되는 리듬감 있는 가슴 흔들림 동작을 추가합니다. **Intensity**는 진폭을 설정하고; **Speed**는 비트 주파수를 조정합니다 (2의 거듭제곱). **Motion Pattern**은 수평 전용(−1), 수직 전용(1), 또는 결합된(0) 흔들림 방향 사이를 혼합합니다.

**Time Pattern**은 리듬을 선택합니다: *Constant*는 모든 비트마다 흔들리며; *Pulse*는 매 비트마다 켜짐/꺼짐을 교차합니다; *Gradual*은 프레이즈 전반에 걸쳐 강도를 쌓아 올리는 미리 정의된 16비트 패턴을 따릅니다. **Abs Mode**는 중립 위치로 되돌아와 보다 과장된 효과를 연출합니다.
