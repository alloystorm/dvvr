# 컨트롤러 입력

입력 설정에서는 키보드/마우스와 게임 컨트롤러(VR 핸드 컨트롤러 포함) 모두에 대한 컨트롤러 입력을 매핑하고 보정할 수 있습니다.


## 옵션

**트랙패드 모드**는 컨트롤러 터치패드의 동작을 변경합니다. 활성화된 경우, 축 값은 패드 버튼이 물리적으로 눌릴 때만 출력됩니다.

**보조 축 사용**은 스틱과 트랙패드를 모두 갖춘 컨트롤러(WMR 컨트롤러 등)의 입력을 읽을 수 있게 합니다.

**게임패드 사용**은 VR 모드로 실행되는 중에도 게임 컨트롤러/게임패드를 사용할 수 있게 합니다.

**그립 입력 사용**은 그립 버튼 입력을 켜거나 끌 수 있습니다. 이를 비활성화하면 실수로 잡거나 그립을 드래그하는 상호 작용을 방지할 수 있습니다.

**그립 임계값** (0.0~1.0)은 그립 버튼의 작동 임계값을 설정합니다. 실수로 트리거되는 것을 방지하기 위해 매우 민감한 컨트롤러(Valve Index 등)에 유용합니다.


## 컨트롤러 매핑

컨트롤러의 다양한 축 및 버튼 매핑을 사용자 지정할 수 있습니다:
* **축(Axes)**은 카메라 이동, 회전, 타임라인 스크러빙 또는 프레임 스텝과 같은 동작에 매핑될 수 있습니다.
* **버튼(Buttons)**은 애니메이션 재생, 다음/이전 모델 로딩, 슬로우 모션 또는 물리 리셋과 같은 동작에 매핑될 수 있습니다.
* **Alt 키(Alt Keys)**는 키보드 키를 컨트롤러 버튼으로 매핑하도록 할당할 수 있습니다.
