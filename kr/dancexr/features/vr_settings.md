## 개요
VR 설정에는 핸드 컨트롤러, UI 동작, 포인터 보정, 성능 옵션을 위한 VR 전용 구성이 포함되어 있습니다. 이 설정은 VR 모드에서 실행할 때만 관련이 있습니다.

## 손
가상 손 렌더링 설정.

* **활성화**: VR에서 가상 손 표시를 전환합니다.
* **그림자 투사**: 가상 손이 씬에서 그림자를 드리울지 여부를 조절합니다.
* **왼손 포즈**: 왼손 컨트롤러의 기본 포즈를 선택합니다.
* **오른손 포즈**: 오른손 컨트롤러의 기본 포즈를 선택합니다.

## UI
VR에서 UI 패널 동작 설정.

* **데스크톱 창 차단**: VR 모드에서 데스크톱 미러 창을 차단하여 데스크톱으로의 화면 출력 렌더링을 생략함으로써 GPU 부하를 줄입니다.
* **UI 자동 복귀**: 활성화 시 UI 패널이 시야 밖으로 벗어나면 부드럽게 시야 안으로 돌아옵니다.
* **UI 거리** (0.5–5m): 사용자로부터 UI 패널이 배치되는 거리를 조절합니다.
* **VR에서 마우스 모드**: 핸드 컨트롤러 없이 마우스를 VR 포인터로 사용할 수 있습니다.
* **마우스 감도**: VR 마우스 모드에서 포인터 감도를 조절합니다.
* **시간 및 FPS**: 손 오버레이에 현재 시간과 프레임 속도를 표시합니다.

## 포인터
VR 상호작용에 사용되는 포인터 레이 보정 설정.

* **방향**: 컨트롤러에 대한 포인터 레이의 방향 각도를 조절합니다.
* **방위**: 포인터의 방위 오프셋을 조절합니다.
* **오프셋**: 포인터 레이 원점의 위치 오프셋을 조절합니다.
* **포인터 업데이트**: 현재 포인터 보정 설정을 적용합니다.

## 포비에이티드 렌더링
포비에이티드 렌더링은 중앙을 선명하게 유지하면서 시야 주변 영역을 낮은 해상도로 렌더링하여 GPU 부하를 줄입니다. 지원되는 하드웨어에서만 표시됩니다.

* **활성화**: 포비에이티드 렌더링을 켜거나 끕니다.
* **레벨** (0–1): 시야 가장자리를 낮은 해상도로 렌더링하는 강도를 조절합니다. 값이 높을수록 주변 이미지 품질을 희생하여 성능이 향상됩니다.
