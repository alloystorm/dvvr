# DanceXR 시작하기

이 가이드는 앱 다운로드부터 모델 불러오기, 모션 재생까지 DanceXR를 처음 사용하는 과정을 순서대로 안내합니다. DanceXR가 처음이어도 걱정하지 마세요. 직관적이고 쉽게 사용할 수 있도록 설계되어 있습니다.

---

## 1. 다운로드 및 설치

DanceXR는 **PC(Windows/Mac)**, **Android**, **iOS**, **Meta Quest**에서 사용할 수 있습니다. 사용하는 플랫폼에 맞는 버전은 다운로드 페이지에서 받으세요.

[**DanceXR 다운로드 →**](/kr/dancexr/download)

사용 가능한 버전은 다음과 같습니다.

| 플랫폼 | 권장 빌드 |
|----------|-----------------|
| **Windows PC** | 균형 잡힌 품질은 **HD**, 레이 트레이싱은 **RT**, 성능 우선은 **LW** |
| **Mac** | **HD** 빌드 — Steam에서 제공되지만 사용자 수가 적어 최근에는 업데이트가 거의 없습니다 |
| **Android** | **LW** 빌드 — Google Play 또는 Itch.io에서 이용 가능 |
| **iOS** | App Store |
| **Meta Quest** | 독립 실행형 빌드 — Itch.io에서 이용 가능 |

**설치 팁:**

- **Windows:** 다운로드한 `.7z` 압축 파일을 원하는 폴더에 풀고 `DanceXR.exe`를 실행하세요.
- **Mac:** Steam을 통해 설치하세요.
- **Android / Quest:** 다운로드한 소스에서 APK를 설치하세요.
- **iOS:** App Store에서 직접 설치하세요.

> 실행 시 문제가 생기면 일반적인 해결 방법이 정리된 [지원 페이지](/kr/dancexr/support)를 확인하세요.

## 2. 콘텐츠 라이브러리 준비

DanceXR를 실행하기 전에 콘텐츠 라이브러리를 준비해 두는 것을 권장합니다. 하지만 모델 파일과 모션 파일을 DanceXR 창으로 직접 드래그 앤 드롭해서 바로 불러올 수도 있습니다. 지금은 이 단계를 건너뛰고, 나중에 직접 모델과 모션을 추가할 준비가 되었을 때 다시 돌아와도 됩니다.

DanceXR는 모델, 모션, 기타 콘텐츠를 **content library** 폴더 안에서 찾습니다. PMX 및 XPS 모델 파일은 이 위치에 복사하면 됩니다.

위치는 플랫폼에 따라 다릅니다.

- **Windows:** DanceXR를 열고 왼쪽 아래의 톱니바퀴 아이콘(시스템 메뉴)을 클릭한 다음, Content Library 아래의 "Show in Explorer"를 선택하세요.
- **Android / Quest:** 2024.3 업데이트 이후 폴더는 저장소의 `/DanceXR/`에 있습니다.
- **iOS:** 기기의 Files 앱에서 "DanceXR" 폴더를 찾으세요.

폴더 구조와 지원 형식에 대한 자세한 내용은 [콘텐츠 라이브러리 가이드](/kr/dancexr/preparecontent)를 참고하세요.

## 3. 기본 UI와 조작

DanceXR를 열면 화면 아래쪽에 메뉴 바가 보여야 합니다. 보이지 않으면 빈 공간을 클릭해 UI 상태를 전환하면서 다시 표시하세요.
VR에서는 UI가 눈앞에 떠 있고, 데스크톱 모드에서는 화면 하단에 고정됩니다. 클릭과 드래그 동작은 마우스를 쓰든 VR 컨트롤러를 쓰든 거의 동일하게 작동합니다. VR에서는 그립 버튼을 누른 채 손을 움직여 UI를 이동할 수 있습니다.

### UI 구성 요소
![DanceXR UI overview](/images/menu.PNG)
- **진행 바:** 화면 하단에 있습니다. 현재 모션 또는 오디오 이름과 진행 상태를 표시합니다. 클릭하면 재생 또는 일시정지하고, 드래그하면 원하는 위치로 이동할 수 있습니다.
- **메뉴 섹션:** 왼쪽에 다섯 개의 아이콘이 있습니다.
  - 시스템 메뉴(톱니바퀴): 일반 설정, 콘텐츠 라이브러리, 지원 링크
  - 환경 메뉴(그림): 무대, 스카이박스, 조명, 카메라 설정 변경
  - 장면 메뉴(무대): 무대와 소품 불러오기, 장면 설정 저장 / 불러오기
  - 오디오/모션 메뉴(음표): 모션 파일과 오디오 파일 불러오기 및 할당
  - 액터 메뉴(사람): 캐릭터 모델 불러오기 및 관리
- **재생 및 채팅 컨트롤:** 오른쪽에 볼륨 슬라이더, 재생목록, 이전/다음 버튼, AI 채팅 토글이 있습니다.

### 장면 조작

불러온 액터의 발 아래에는 노란 원이 표시됩니다.
- 원을 클릭하면 액터 메뉴가 열립니다
- 드래그하면 액터를 무대 위에서 이동할 수 있습니다
- 드래그하는 동안 마우스 휠을 사용하면 액터를 회전할 수 있습니다

### 표시 모드 전환

빈 공간을 클릭하면 다음 모드 사이를 전환할 수 있습니다.
- **UI 모드:** 모든 메뉴와 컨트롤이 표시됩니다.
- **컨트롤 모드:** 메뉴는 숨겨지지만 액터 원은 보입니다.
- **몰입 모드:** 메뉴와 원이 모두 숨겨져 가장 몰입감 있는 화면이 됩니다.

## 4. 모델 불러오기

### 아직 콘텐츠 라이브러리에 없는 경우

문제 없습니다. 모델 파일은 DanceXR 창에 직접 드래그 앤 드롭해서 바로 불러올 수 있습니다. 지원 형식은 `.pmx`, `.xps`, `.mesh`, `.mesh.ascii`입니다. 파일을 드롭하면 액터 메뉴에 표시됩니다.

### 콘텐츠 라이브러리에서 불러오기

액터 아이콘을 클릭하고 "Load Model"을 선택하세요. `actors` 폴더에 넣어 둔 모델 목록이 표시됩니다. 기본적으로 새 모델은 현재 무대에서 선택된 모델을 교체합니다. 교체 대신 추가하려면 모델 이름 옆의 "+" 아이콘을 클릭하세요. 이 기능은 유료 버전에서 사용할 수 있습니다.

## 5. VMD 모션 불러오기

### 드래그 앤 드롭

오디오 파일을 DanceXR 창에 드래그하면 즉시 불러옵니다. 지원 형식은 WAV와 OGG입니다.
VMD 또는 BVH 형식의 모션 파일을 DanceXR 창에 드래그하면 해당 모션이 불러와집니다. 현재 선택된 액터에 아직 모션이 할당되어 있지 않다면 그 액터에 자동으로 할당됩니다.

### 콘텐츠 라이브러리

오디오/모션 아이콘을 클릭하고 "Load Audio / Motion"을 선택하세요. `motions` 폴더에 넣어 둔 모션 목록이 표시됩니다.

모션과 오디오는 자동으로 "dance sets"로 묶입니다. 불러온 뒤에는 진행 바에 오디오 이름이 표시되고, 댄스 메뉴에 모션 목록이 나타납니다. 모션을 할당하려면 해당 모션을 선택한 뒤 "Assign To"를 선택하거나, 액터 메뉴를 열고 불러온 모션 중 하나를 선택하세요.

모션 파일 정리에 대한 자세한 내용은 [콘텐츠 라이브러리 가이드](/kr/dancexr/preparecontent#motion-files)를 참고하세요.
