# 縮放與偏移

調整角色在場景中的整體尺寸、位置和旋轉。

**模型縮放** 使用 2 的次方指數縮放，因此微小變動可以產生精細的調整。**統一高度** 會將所有角色縮放到一致的平均人形高度，而與其原始模型比例無關——當您想保留模型預計尺寸時，請禁用此選項。**地面偏移** 會讓角色相對於地面上下移動。

**旋轉** 讓角色繞 Y 軸旋轉。**X 偏移** 和 **Z 偏移** 在水平方向上移動位置。**吸附** 設定了在場景中拖動角色時的增量大小——設為零將完全禁用吸附功能。

預設值包含 *統一真人尺寸*、*微縮*、*巨型* 和 *原始*。
