# 物理

配置包含內建剛體和關節定義（髮型、裙子、飾品）的 PMX 模型物理效果。

**物理模式** 選擇引擎：*自動* 使用系統預設；*PhysX* 運行 Unity 的關節物理；*XPBD* 使用基於粒子的模擬；*關閉* 完全禁用 PMX 物理，並將 XPS 物理工具（髮型物理、衣物物理等）作為替代方案顯示。

**可視化骨體** 和 **可視化關節** 將物理形狀和關節約束渲染為線框飾物。嵌套的 **粒子屬性** 面板定義了全局的 XPBD 設置，例如重力、剛度和阻尼。

**線性運動** 和 **角運動** 調整 PhysX 關節驅動響應——即身體追隨動畫目標的積極程度。**選項** 顯示了諸如解算器迭代次數和子步數等高級 PhysX 參數。**自動重置閾值** 在速度超過限制時將骨骼恢復到動畫位置，防止物理模擬失控。

每個 **群組** 部分對應於 PMX 文件中定義的一個物理關節組，具備每組的啟用/禁用、重力和碰撞設置。

**XPS 工具** 部分提供了快速連結，指向當關閉 PMX 物理時可用的替代物理系統。
