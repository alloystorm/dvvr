# 網格轉布料

將角色上的任何渲染網格轉換為基於粒子的布料模擬。每個網格都會獲得自己的設定面板，用於選擇固定頂點（錨定布料的骨骼）和模擬參數。

**漸進式啟用 (Gradual Enable)** 會在設定的秒數內淡入模擬，防止網格突然「彈入」到位。嵌套的 **Particle Props** 面板定義了諸如剛度、阻尼和重力乘數等全域物理屬性，這些屬性將應用於所有轉換的網格。

# 子組件

## 粒子屬性

包含用於 PMX 物理模式中 XPBD 物理系統的粒子尺寸、質量、阻尼、重力、摩擦和碰撞層。
