# 假陰莖

<!-- TODO: confirm exact UI labels and configuration. Drafted from related Sex Overlay docs and the prop-attachment system. -->

設定將假陰莖道具附加到角色上。可單獨與單一角色使用，或在未載入男性角色時作為配對動作的男性部分使用。

---

## 設定

1. 在相關角色上啟用假陰莖。
2. 選擇附加骨骼/位置 — 通常錨定在骨盆。
3. 調整大小和偏移量。
4. 可選：配置與[性動態](sex_motion_3) 或[自動更新](autoupdate) 綁定的動畫行為（推擊模式）。

詳細的附加配置位於[性疊加與假陰莖配置](smo_config)。

## 與動態的配對

- 使用[性動態3](sex_motion_3) — 假陰莖可以在配對動作中填補男性角色。動作的接觸框系統會將假陰莖對齊女性的接觸點。
- 使用[自動更新](autoupdate) — 從音訊電平或節拍驅動推擊模式，實現音樂同步動作。
