# 物理學

場景的全局物理引擎選擇、時間步長設定和重力。


## 預設引擎

**PMX 預設** 設定了內建剛體和關節（髮型、裙子、配件）的 PMX 模型所使用的物理引擎。請為 Unity 的基於關節的物理使用 *PhysX*，為基於粒子的模擬使用 *XPBD*，或使用 *關閉* 來禁用，並將這些骨骼釋放給 XPS 自訂物理工具使用。

**自訂物理預設** 為 XPS 工具（髮型物理、布料物理、Softbody 等）設定引擎。XPBD 對布料更穩定；PhysX 在某些硬體上可能更快。


## PhysX 設定

**重力** 設定應用於所有剛體向下的力量。將其設定為零可在無重力舞台使用。**禁用碰撞** 可防止 XPS 粒子系統之間發生碰撞，這可以減少多個 Softbody 互動時的抖動。


## XPBD 設定

包含 XPBD 引擎的模擬參數——詳情請參閱 XPBD 設定子資料。


## 時間

**時間尺度** 會獨立於動畫播放速度來減速或加速物理模擬。較低的值使粒子和 Softbody 呈現慢動作。**物理 FPS** 將模擬鎖定到固定的時間步長：較低的值可減少 CPU 負載，較高的值以犧牲性能為代價提高穩定性。*靈活* 讓模擬以渲染幀率運行。


# 子組件

## XPBD 設定

包含 XPBD 求解器設定：子步、迭代次數、碰撞間隔、自碰撞、物體碰撞、地表摩擦、膨脹半徑和平行求解。
