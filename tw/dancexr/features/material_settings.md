## 材質如何組織<a id="material-category"></a>

一個模型可能有數百種不同的材料。為了更有效地管理它們，我們以以下方式組織材料：

* 紋理組
  為了使材料列表更易管理，我們通過它們使用的紋理文件將它們分組。大多數情況下，皮膚、服裝、配飾和頭髮都使用不同的紋理貼圖。因此，按照紋理分組為不同目的提供了良好的區分。

* 材料類別
  除了紋理分組外，我們還嘗試通過檢查材料和紋理名稱來確定每個材料的目的。根據這些目的，我們創建了更易管理材料的類別。
  這些類別包括
  * 皮膚：身體、臉部、手臂和腿部屬於此類別，它們自動分配皮膚著色器以獲得最佳外觀。
  * 頭髮：頭髮材料也有自己的著色器以獲得最佳外觀。
  * 眼睛：眼睛具有不同的默認值，如更高的光澤。
  * 嘴唇：嘴唇也分配了與標準材料不同的值。
  * 不透明：不屬於前述類別的不透明材料。
  * 透明：不屬於其他類別的透明材料。
  * 自定義：默認為空，但您可以將材料分配給此類別並對其進行自定義調整。

## 進行調整

### 材料列表<a id="material-list"></a>

您可以從演員菜單中找到材料列表。只需轉到“材料” / “材料列表”，您將看到所有材料按其紋理組列出。

您可以在材料列表中進行的調整：
* 組特定控件：
  * 切換：控制整個組的可見性
  * 覆蓋：如果您希望將相同更改應用於組中的所有材料，請打開“覆蓋”並在其中進行調整。
  * 共享材料：打開此選項以強制整個組使用來自第一項的相同材料。這可能會通過減少多個材料的開銷來提高性能。請注意，有時個別材料具有不同的設置，使用共享材料在這種情況下可能會產生意外結果。
* 個別材料控制：
  * 更改類別：更改默認類別
  * 透明度：在自動、不透明和透明之間進行更改
  * 透明預通道：這是一種解決α排序問題的技術。但它的缺點是只允許第一層顯示。如果不需要，可以關閉它。
  * 透明預通道閾值：高於閾值的α值將不使用預通道。
  * 表面：調整顏色、光滑度、金屬和發光
  * 陰影：選擇是否投下陰影
  * 陰影閾值：只有α值高於閾值時才會投下陰影。

### 材料類別控件

* 皮膚
  使用皮膚材料，您可以控制細節圖比例、厚度和汗水效果。
* 頭髮
  頭髮著色器具有細節圖比例和各向異性（高光變得定向）控制

## 紋理

一個材料可以攜帶無限數量的紋理（自 2025.9 起），每種紋理都在表面外觀的各個方面扮演不同的角色。材料設置中的紋理部分列出了所選材料上的每種紋理；點擊紋理可以檢查它，如果 DanceXR 的自動檢測判斷有誤，您也可以重新分配其目的。

DanceXR 會根據紋理文件名自動分配紋理類型和通道映射。大多數情況下您不需要更改任何內容——此部分剩餘內容是為您在需要進行更改的情況預留的。

### 基本紋理類型

對材料單一角色進行映射的紋理：

| 類型 | 作用 |
|---|---|
| 顏色 | 表面主體顏色（也稱為漫反射/反照率/底色）。 |
| 凹凸 | 通過改變表面法線來添加表面浮雕——皺紋、接縫、織物紋理。 |
| 反射光 | 定義傳統反射光工作流程中的高光顏色和銳利度。DanceXR 使用 PBR 工作流程，因此反射光數據會被重定向到金屬度和粗糙度貼圖。 |
| 自發光 | 使表面看起來像發出光線。用於發光效果、屏幕、眼睛等。 |
| 透明度 (Alpha) | 控制材料的透明程度。 |

### 進階紋理類型

一些紋理將多個角色打包到單一圖像的通道（R、G、B、A）中。DanceXR 支持兩種：

- **細節** — 一種平鋪紋理，用於在底色上添加精細的細節（皮膚毛孔、織物紋理）。
- **遮罩** — 定義表面哪些區域會受到哪些屬性的影響（金屬度、粗糙度、遮蔽、細節）。

對於這些，您需要將紋理的每個通道分配給一個特定的目的。可用的通道目的包括：

| 目的 | 用於 | 控制內容 |
|---|---|---|
| 金屬度 | 遮罩貼圖 | 表面每個像素的金屬度。 |
| 遮蔽 | 遮罩貼圖 | 環境遮蔽——縫隙中的柔和陰影。 |
| 光滑度 | 遮罩貼圖 | 表面有多光滑（像鏡子一樣）。 |
| 粗糙度 | 遮罩貼圖 | 光滑度的反面——為使用粗糙度而非光滑度工作流程編寫紋理提供。 |
| 凹凸 X | 細節貼圖 | 細節法線圖的 X 軸。 |
| 凹凸 Y | 細節貼圖 | 細節法線圖的 Y 軸。 |
| 細節遮罩 | 遮罩貼圖 | 限制應用細節紋理的位置。 |

總而言之，單一的遮罩紋理的紅色通道可能會驅動金屬度，綠色通道用於遮蔽，藍色通道用於平滑度——DanceXR 分別展示了每個通道，您可以單獨重新指向它們。

### 檢查紋理

在紋理子菜單中：

- 紋理本身顯示在面板頂部。
- 點擊紋理可將其拆分成單個通道（R、G、B、A），讓您查看每個通道承載了什麼——當您搞不清楚打包的遮罩紋理是如何創建時，這特別有用。
- 在預覽圖下方，選項允許您更改紋理的類型分配和通道目的。

這是了解不熟悉紋理用途的最快方法：打開它，查看通道，如果需要則重新分配。
