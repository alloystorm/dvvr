# 爛布娃娃

使用基於粒子的物理模擬取代演員的動畫骨架。身體會失去張力，並對重力和外部力量做出反應。

**彈簧力** 和 **阻尼**（兩者皆為對數）控制爛布娃娃試圖返回其動畫姿勢的積極程度——低值會使其表現得鬆軟，高值會使其更接近動畫。**重力** 用於開關對爛布娃娃的重力拉力。

**鎖定左/右腳**、**鎖定頭部** 和 **鎖定左/右手** 將特定的身體部位固定在動畫位置，使其不至於下垂——這對於在身體其他部位鬆弛時，保持雙腳站立非常有用。

**細微調整** 設定從 0（完全鬆弛）到 1（完全僵硬）的每骨骼力係數。調整個別身體部位，如腿、膝、手臂、肩膀、軀幹、頸部和頭部，以創造局部僵硬度——例如，保持軀幹堅實，同時讓四肢懸垂。
