# 腳部調整

確保腳部能動態適應地面高度，達到逼真的擺放和動作。

**地面偏移** 調整垂直參考點。**腳部貼地** 面板控制 **強度** (向地面拉力)、**足底高度** (腳趾到地面的距離)、**身體抬升** (慣性點校正)、**接觸彎曲足** (腳趾屈曲)、**接觸旋轉** (旋轉慣性)、**跳躍偵測** (懸空抑制)，以及 **跳躍高度** 閾值。

**虛擬鞋子** 面板可視化腳部接觸，包含 **內側**、**外側**、**前**、**後** 偏移、**鞋子距離**、**腳跟頂部** 高度、**模式** (線框、不透明、鍍鉻、實心玻璃、薄玻璃)、**平滑度**，和 **顏色**。

**腳跟** 面板模擬高跟鞋造型，包含 **高跟鞋** 高度、**限制角度**、**高跟鞋腳趾彎曲**，以及用於動畫混合的 **高跟鞋混合**。

**減少滑動** 使用 **距離閾值** 和 **阻尼** 控制摩擦力。**足部角度** 增加手動的 **X**、**Y** 和 **Z** 偏移。
