# 腳部調整

確保腳部能動態適應地面高度，達到逼真的擺放和動作。

**地面偏移** 調整垂直的地面參考點。

**保持足部貼地** 面板用於將腳固定在地面上：
**強度**（整體強度，0 表示禁用）、**足底高度** (腳趾到地面的間隙)、**跳躍高度** (懸空偵測閾值，用來暫停校正)、**身體平滑度**和**腳部平滑度** (用於身體和單個腳部的反應平滑化)，**最大足部傾斜角度** (為在地面上設定接觸點，足部可傾斜的程度)，以及 **延伸彎曲度** (當腿無法放下時，保持接觸所需的額外腳趾彎曲)。

**腳跟** 面板將足部抬高至**高跟鞋**姿態。

**減少滑動** 用於固定已著地的腳底以停止遊走：啟用此功能，然後設定**強度**（鎖定程度）和**滑動距離**（釋放前可漂移的距離，以避免過度拉伸腿部）。

**足部角度** 新增手動的 **X**、**Y** 和 **Z** 偏移。
