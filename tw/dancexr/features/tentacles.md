# 觸手

會生成並模擬附加到演員身上的程序化觸手資源。每條觸手都是由 XPBD 物理引擎驅動的、具備顫動、捲曲和吸引行為的分段網格。

**觸手數量** 和 **長度** 設定了數量和尺寸。**生成** 面板控制叢集區域：**區域形狀**（圓形或矩形）、**區域大小**、**長寬比** 和 **長度分佈** 用於變化。**半徑**、**錐度** 和 **尖端半徑** 塑造每條觸手的粗細曲線。

**行為** 面板驅動動畫：**顫動速度**、**範圍** 和 **延遲** 營造自然的擺動；**捲曲範圍** 和 **捲曲衰減** 添加向尖端淡化的螺旋捲曲。**吸引力** 和 **吸引偏移** 將觸手尖端拉向追蹤的身體位置。**動作** 將運動與性動作系統同步；**動作範圍** 縮放反應。**回縮距離** 控制觸手收回的距離。

**材質** 和 **X光** 子面板控制外觀和橫截面渲染。**觸手資源** 定義了全局物理參數。更改數量或生成參數後，點擊 **重建觸手** 以重新生成網格。
