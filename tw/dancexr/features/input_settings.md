# 控制器輸入

輸入設定允許您為鍵盤/滑鼠和遊戲控制器（包括 VR 手柄控制器）對控制器輸入進行映射和校準。


## 選項

**跟蹤墊模式**會改變控制器觸控板的行為。啟用此功能後，軸向數值僅在實體按下踏板按鈕時輸出。

**使用次級軸向**允許讀取具有滾筒式滑鼠和跟蹤墊（例如 WMR 控制器）的控制器輸入。

**使用遊戲手把**即使在 VR 模式下也能啟用遊戲控制器/遊戲手把。

**使用握持輸入**開關握持按鈕的輸入開關。將其關閉可防止意外抓取或握持拖曳互動。

**握持門檻** (0.0 到 1.0) 設定了握持按鈕的觸發門檻。對於高靈敏度控制器（例如 Valve Index）非常有用，可以防止誤觸。


## 控制器映射

您可以自定義控制器上各種軸向和按鈕的映射：
* **軸向**可以映射到攝影機移動、旋轉、時間軸捲動或幀步進等動作。
* **按鈕**可以映射到播放動作、載入下/上模型、慢動作或重置物理效果等動作。
* **Alt 鍵**可用於分配映射將鍵盤按鍵對應到控制器按鈕。
