# 光球

生成附加到角色骨骼上的物理驅動發光球體。
**形狀** 選擇球體、菱形或水晶網格。**尺寸** 縮放球體。切換連接點：**手部**、**髖部** 和 **軀幹**。

**樣式** 面板使用預設（Glow, Reflective, Crystal）控制外觀，包括顏色、光澤、發光、金屬度、強度、折射率、陰影和尾跡。使用 **舞台顏色** 同步場景照明或使用 **配合節拍閃爍** 實現音訊反應脈衝。

**物理** 面板配置彈簧關節行為，包含預設（Spring, Hanging, Floating）、重力、碰撞、質量、距離、彈簧力、阻尼和速度限制。

# 子組件

## 樣式

光球外觀的巢狀配置。預設：**發光** (emissive)、**反射** (metallic)、**水晶** (refractive)。**使用舞台顏色** 同步場景照明。**配合節拍閃爍** 將強度脈衝至音訊。控制 **顏色**、**光澤**、**發光**、**半徑**、**金屬度**、**強度**、**折射**、**投射陰影** 和 **尾跡** 長度。

## 物理

光球物理行為的巢狀配置。預設：**彈簧** (無重力/碰撞)、**懸掛** (重力 + 碰撞)、**漂浮** (無重力/碰撞)。**重力** 啟用向下力。**碰撞** 啟用物理碰撞。**質量**、**距離**、**彈簧力** 和 **阻尼** 配置彈簧關節。**最小/最大速度** 限制速度。
