# 遠端存取

透過遠端控制，Android 應用程式可作為無線控制器，用於操作位於相同區域網路（LAN）上的其他設備上運行的 DanceXR。您可以在手機或平板電腦更改場景、動作和設定，而主流程則在更大的螢幕上運行。


## 必要條件

* 安裝了 DanceXR Android 應用程式的 Android 設備。
* 一台在相同區域網路（LAN）上運行的 PC、Quest 或其他 DanceXR 主機。
* 該網路必須允許本地設備的發現（不能有客戶端隔離）。


## 連接方法

1. 在主機設備上啟用 **遠端存取**。
2. 設定 **連接埠號碼** 和安全的 **密碼**。
3. 開啟 Android 應用程式，在主選單中選擇 **遠端控制**，然後從發現的設備中選擇主機。
4. 當系統提示時，輸入密碼以同步選單和觸控板控制功能。
