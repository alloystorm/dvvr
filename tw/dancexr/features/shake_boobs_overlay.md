# 搖晃胸部覆蓋層

添加由音樂節奏驅動的節奏性胸部搖晃動作。
**強度** 設定振幅；**速度** 縮放節拍頻率（2的次方）。**動作模式** 在純水平 (−1)、純垂直 (1) 或組合 (0) 搖晃方向之間混合。

**時間模式** 選擇節奏：*Constant* 在每個節拍時搖晃；*Pulse* 每隔一個節拍交替開/關；*Gradual* 遵循預定義的 16 拍模式，並在樂句中增強強度。**Abs Mode** 在中性位置彈回，以產生更誇張的效果。
