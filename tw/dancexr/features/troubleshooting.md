# 疑難排解

修復常見的動畫和骨架綁定問題。**應用身體旋轉至中心**會將臀部和軀幹的旋轉轉移到中心骨骼。**扭曲校正**用於減少手臂和腿部的關節扭曲，並設有單獨的**上臂**、**前臂**和**腿部扭曲**滑塊。**肘軸**用於調整手肘關節的旋轉軸。**手部縮放**用於調整手部大小。**BVH拇指動態**用於減少用於BVH動畫的拇指移動。**限制頸部/頭部旋轉**會限制上半身的旋轉。**重置過渡**從標準姿態執行平穩的物理重置。**跳過運動學更新**會停用未動畫化的運動學骨骼的更新。
