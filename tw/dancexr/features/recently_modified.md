# 最近修改的設定

這是可快速存取的、您最近更改的每個角色設定清單。它允許您無需在角色菜單層級結構中導航，即可重新開啟相同的對話框。

可在角色菜單中找到——參閱 [角色菜單與工具](/dancexr/features/actor_tools#actor-menu)。

---

## 存在的原因

角色菜單包含許多區塊——材質、物理、服裝、動作、行為。當您正在迭代單一功能時（調整頭髮物理、比較兩種皮膚材質設定），直接跳回最常使用的對話框比再次深入層級更快速。

「最近修改的設定」提供了這個快捷方式。隨著您更改設定，清單會自動更新；最近一次的更改會顯示在頂部。

---

## 如何使用

<!-- TODO: confirm exact UI behavior. Below is what is implied by the actor_tools menu reference. -->

1. 開啟角色菜單（點擊角色的選擇圓盤）。
2. 點擊菜單頂部的 **最近修改**。
3. 清單顯示了您為此角色調整的最新的設定，最新的會排在最前面。
4. 點擊一個項目即可跳轉到該對話框。

此清單是針對每個角色的，重新加載模型或場景時會重設。
